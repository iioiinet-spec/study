회사에서 출발해 모든 장소를 한 번씩 방문하고 다시 회사로 돌아오는 최소 비용 경로를 찾아봅니다.

1. 문제를 쉽게 이해하기

해밀턴 순환회로

모든 도시를 딱 한 번씩 방문하고 다시 출발 도시로 돌아오는 길입니다.

정올 1681의 목표

1번 회사에서 출발해서 모든 장소를 방문한 뒤 1번으로 돌아오는 최소 비용을 구합니다.

주의할 점

비용표의 0은 길이 없음입니다. 또한 i→j 비용과 j→i 비용이 다를 수 있습니다.

초등학생 버전: 택배 아저씨가 회사에서 출발해 모든 집을 한 번씩만 들르고 다시 회사로 돌아와야 합니다. 여러 길 중에서 기름값이 가장 적게 드는 길을 찾는 문제입니다.

5. 핵심 알고리즘 설명

  1. 1번 회사에서 출발합니다.
  2. 아직 방문하지 않은 도시 중에서 갈 수 있는 도시를 하나 선택합니다.
  3. 그 도시를 방문 표시하고 비용을 더합니다.
  4. 모든 도시를 방문했다면 다시 1번으로 돌아갈 수 있는지 확인합니다.
  5. 돌아갈 수 있으면 총비용으로 정답을 갱신합니다.
  6. 현재 비용이 이미 찾은 정답보다 크거나 같으면 더 볼 필요가 없으므로 가지치기합니다.
  7. 다른 경로도 확인하기 위해 방문 표시를 다시 해제하고 돌아갑니다. 이것이 백트래킹입니다.

6. 정답 C 코드

#include <stdio.h>

#define INF 987654321

int N;
int cost[14][14];
int visited[14];
int answer = INF;

void dfs(int now, int count, int sum) {
    int next;

    // 가지치기: 이미 현재 비용이 정답보다 크거나 같으면 중단
    if (sum >= answer) {
        return;
    }

    // 모든 장소를 방문한 경우
    if (count == N) {
        // 현재 위치에서 회사 1번으로 돌아갈 수 있어야 함
        if (cost[now][1] != 0) {
            if (sum + cost[now][1] < answer) {
                answer = sum + cost[now][1];
            }
        }
        return;
    }

    // 다음 방문할 장소 선택
    for (next = 2; next <= N; next++) {
        if (visited[next] == 0 && cost[now][next] != 0) {
            visited[next] = 1;
            dfs(next, count + 1, sum + cost[now][next]);
            visited[next] = 0;
        }
    }
}

int main() {
    int i, j;

    scanf("%d", &N);

    for (i = 1; i <= N; i++) {
        for (j = 1; j <= N; j++) {
            scanf("%d", &cost[i][j]);
        }
    }

    visited[1] = 1;
    dfs(1, 1, 0);

    printf("%d\n", answer);

    return 0;
}

7. 수업용 정리

DFS

한 길을 끝까지 가보고, 막히면 돌아와서 다른 길을 시도합니다.

백트래킹

방문 표시를 했다가, 다른 경우를 보기 위해 다시 방문 해제합니다.

가지치기

이미 비용이 너무 크면 더 가도 정답이 될 수 없으므로 빨리 포기합니다.

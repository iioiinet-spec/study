정올 #1516 단어 세기

문장을 입력받아 공백으로 단어를 나누고, 같은 단어가 몇 번 나왔는지 세는 방법을 배웁니다.

1. 문제를 자세히 이해하기

입력

여러 줄의 문장이 들어옵니다.

종료 조건

한 줄이 END이면 입력을 끝냅니다.

해야 할 일

각 줄마다 단어를 나누고, 같은 단어가 몇 번 나왔는지 출력합니다.

단어는 공백으로 구분합니다. 예를 들어 apple banana apple은 apple 2개, banana 1개입니다.

2. 풀이 핵심

  1. 한 줄을 fgets()로 입력받습니다.
  2. 줄 끝의 개행 문자 \n을 제거합니다.
  3. 입력 줄이 END인지 확인합니다.
  4. strtok(line, " ")으로 공백 기준 단어를 나눕니다.
  5. 단어들을 배열에 저장합니다.
  6. qsort()로 단어를 사전순 정렬합니다.
  7. 정렬하면 같은 단어가 붙어 있으므로 앞에서부터 개수를 셉니다.

3. 인터랙티브로 단어 세기 보기

1단계: 입력 줄

2단계: 단어 분리

3단계: 정렬된 단어

4단계: 단어별 개수

실행을 누르면 단어가 나뉘고 정렬된 뒤 개수가 계산됩니다.

END가 나오면 그 줄부터는 처리하지 않습니다.

처리한 문장 수

2

현재 줄

1

현재 단계

준비

4. 출력 결과

apple : 2
banana : 2
orange : 1
cat : 2
dog : 1

5. 단계별 처리 표

단계문장 번호작업내용설명

6. 코드 작성 방법

  1. line 배열을 만들어 한 줄 전체를 저장합니다.
  2. words 2차원 배열에 단어들을 저장합니다.
  3. fgets()를 이용해 한 줄씩 읽습니다.
  4. line[strcspn(line, "\n")] = '\0';으로 줄바꿈을 없앱니다.
  5. strcmp(line, "END") == 0이면 반복문을 종료합니다.
  6. strtok()으로 단어를 하나씩 꺼냅니다.
  7. 단어를 strcpy()로 words 배열에 복사합니다.
  8. qsort()로 단어를 정렬합니다.
  9. 정렬된 단어를 앞에서부터 보면서 같은 단어가 몇 개인지 셉니다.
  10. 단어와 개수를 출력합니다.

7. 초등학생도 이해하는 설명

문장을 단어 카드로 자릅니다. 그런 다음 같은 단어 카드를 서로 붙도록 정렬합니다. 그러면 같은 단어들이 한 줄로 모이기 때문에 몇 장인지 쉽게 셀 수 있습니다.

apple banana apple orange banana

단어 카드:
apple / banana / apple / orange / banana

정렬 후:
apple / apple / banana / banana / orange

개수:
apple 2개
banana 2개
orange 1개

8. 정답 C 코드

#include <stdio.h>
#include <string.h>
#include <stdlib.h>

char line[10001];
char words[1000][101];

int compare(const void *a, const void *b) {
    return strcmp((char *)a, (char *)b);
}

int main() {
    while (1) {
        fgets(line, sizeof(line), stdin);

        line[strcspn(line, "\n")] = '\0';

        if (strcmp(line, "END") == 0) {
            break;
        }

        int n = 0;

        char *token = strtok(line, " ");

        while (token != NULL) {
            strcpy(words[n], token);
            n++;
            token = strtok(NULL, " ");
        }

        qsort(words, n, sizeof(words[0]), compare);

        for (int i = 0; i < n; i++) {
            int cnt = 1;

            while (i + 1 < n && strcmp(words[i], words[i + 1]) == 0) {
                cnt++;
                i++;
            }

            printf("%s : %d\n", words[i], cnt);
        }
    }

    return 0;
}

정올 #1009 각 자리수의 역과 합

수를 뒤집는 방법과 각 자리 숫자의 합을 구하는 방법을 직접 실험하면서 이해해 봅시다.

1문제 핵심

양의 정수를 입력받아 역으로 뒤집은 수와 각 자리 숫자의 합을 출력합니다.
여러 개의 수가 입력될 수 있으며, 0이 입력되면 종료합니다.
0은 유효한 데이터가 아니므로 출력하지 않습니다.
중요 포인트
453 → 뒤집은 수 354, 자릿수 합 4 + 5 + 3 = 12
35603200 → 뒤집으면 00230653이지만, 정수로 출력하면 앞의 0이 사라져서 230653이 됩니다.

2인터렉티브 입력

여러 줄에 숫자를 입력해 보세요. 마지막에 0을 넣으면 그 전까지만 처리합니다.

3출력 결과

실행하기 버튼을 누르면 결과가 표시됩니다.

4첫 번째 수 처리 과정

입력된 첫 번째 양의 정수를 기준으로, 일의 자리부터 하나씩 꺼내는 과정을 보여줍니다.

digit = n % 10
reverse = reverse * 10 + digit
sum = sum + digit
n = n / 10

5코드 작성 방법

  1. while문으로 계속 입력을 받습니다.
  2. 입력값이 0이면 break로 반복을 끝냅니다.
  3. 각 숫자마다 reverse와 sum을 0으로 초기화합니다.
  4. n % 10으로 가장 오른쪽 숫자를 꺼냅니다.
  5. 뒤집은 수는 reverse = reverse * 10 + digit으로 만듭니다.
  6. 자릿수 합은 sum += digit으로 누적합니다.
  7. n /= 10으로 이미 사용한 일의 자리를 제거합니다.
  8. n이 0이 되면 reverse와 sum을 출력합니다.

6초등학생도 이해하는 설명

숫자 453을 뒤에서부터 읽으면 3, 5, 4입니다.
이 숫자들을 차례대로 붙이면 354가 됩니다.

그리고 각 자리 숫자를 모두 더하면
4 + 5 + 3 = 12입니다.

그래서 453의 출력은 354 12입니다.

7정답 C 코드

#include <stdio.h>

int main() {
    int n;

    while (1) {
        scanf("%d", &n);

        if (n == 0) {
            break;
        }

        int reverse = 0;
        int sum = 0;

        while (n > 0) {
            int digit = n % 10;
            reverse = reverse * 10 + digit;
            sum += digit;
            n /= 10;
        }

        printf("%d %d\n", reverse, sum);
    }

    return 0;
}

정올 #1092 제곱수 출력

X의 Y제곱을 20,091,024로 나눈 나머지를 빠른 제곱으로 구하는 문제입니다.

1. 문제를 자세히 이해하기

정답 = XY % 20091024

입력

두 정수 X, Y가 주어집니다.

X Y

출력

X를 Y번 곱한 값을 20091024로 나눈 나머지를 출력합니다.

X^Y % 20091024

주의

XY를 직접 계산하면 숫자가 너무 커져서 일반 변수에 저장할 수 없습니다.

핵심은 큰 수를 직접 만들지 않고, 곱할 때마다 나머지를 구하면서 값을 작게 유지하는 것입니다.

2. 왜 그냥 곱하면 안 될까?

예를 들어 2100만 해도 아주 큰 수입니다. Y가 더 커지면 C언어의 정수형 변수에 담을 수 없습니다.

2^10 = 1024
2^30 = 1073741824
2^100 = 아주 큰 수

하지만 문제는 전체 값이 아니라 나머지만 원합니다.

(a * b) % MOD = ((a % MOD) * (b % MOD)) % MOD

그래서 매번 곱한 뒤 바로 % 20091024를 해주면 숫자가 너무 커지는 것을 막을 수 있습니다.

3. 빠른 제곱 원리

Y번을 하나씩 곱하면 시간이 오래 걸립니다. 그래서 지수를 반씩 줄입니다.

2^10 = (2^5)^2
2^5 = 2 × (2^2)^2

지수 변화: 10 → 5 → 2 → 1 → 0
  1. result를 1로 시작합니다.
  2. x는 MOD로 나눈 값으로 시작합니다.
  3. y가 홀수이면 result에 x를 곱합니다.
  4. x는 매번 제곱합니다.
  5. y는 매번 2로 나눕니다.
  6. 모든 곱셈 뒤에는 MOD로 나머지를 구합니다.

4. 인터랙티브 계산 과정

현재 계산 상태

result = 1, x = 2, y = 10

지수 y가 줄어드는 흐름

실행을 누르면 빠른 제곱 과정이 한 단계씩 표시됩니다.

MOD는 20091024로 고정입니다.

정답

1024

현재 result

1

현재 x

2

현재 y

6. 코드 구현 방법

  1. MOD 값을 상수로 만듭니다.
  2. 곱셈 중 값이 커질 수 있으므로 long long을 사용합니다.
  3. power(x, y) 함수를 만듭니다.
  4. result를 1로 초기화합니다.
  5. x를 먼저 x % MOD로 줄입니다.
  6. while(y > 0) 반복문을 사용합니다.
  7. y가 홀수이면 result = result * x % MOD를 합니다.
  8. x = x * x % MOD로 밑을 제곱합니다.
  9. y = y / 2로 지수를 절반으로 줄입니다.
  10. 반복이 끝나면 result를 출력합니다.

7. 초등학생도 이해하는 설명

같은 숫자를 아주 많이 곱해야 하는데, 하나씩 곱하면 오래 걸립니다. 그래서 “두 번 곱한 것”, “네 번 곱한 것”, “여덟 번 곱한 것”처럼 묶어서 빠르게 계산합니다.

2^13 = 2^8 × 2^4 × 2^1

13의 이진수는 1101
즉, 필요한 묶음은 8, 4, 1입니다.

빠른 제곱은 바로 이 묶음을 자동으로 찾아서 곱하는 방법입니다.

8. 정답 C 코드

#include <stdio.h>

#define MOD 20091024

long long power(long long x, long long y) {
    long long result = 1;

    x = x % MOD;

    while (y > 0) {
        if (y % 2 == 1) {
            result = (result * x) % MOD;
        }

        x = (x * x) % MOD;
        y = y / 2;
    }

    return result;
}

int main() {
    long long x, y;

    scanf("%lld %lld", &x, &y);

    printf("%lld\n", power(x, y));

    return 0;
}

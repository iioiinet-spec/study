1. 문제 이해

정올 1157번 버블정렬은 숫자들을 버블 정렬로 오름차순 정렬하면서, 각 단계가 끝날 때마다 배열 전체를 출력하는 문제입니다.

주의: 최종 정렬 결과만 출력하면 틀립니다. 바깥 반복문 한 번이 끝날 때마다 배열 상태를 출력해야 합니다.

예시

입력
4
62 23 32 15

출력
23 32 15 62
23 15 32 62
15 23 32 62

2. 버블정렬 개념

버블 정렬은 서로 옆에 있는 두 숫자를 비교해서, 왼쪽 숫자가 더 크면 서로 바꾸는 정렬 방법입니다.

비교arr[j]와 arr[j+1]을 비교합니다.
교환왼쪽 값이 더 크면 두 값을 바꿉니다.
한 단계 완료한 바퀴가 끝나면 가장 큰 값이 맨 뒤로 갑니다.
출력정올 1157은 각 단계 완료 후 배열을 출력합니다.

2.1 핵심 코드 구조

for (int i = 0; i < n - 1; i++) {
    for (int j = 0; j < n - 1 - i; j++) {
        if (arr[j] > arr[j + 1]) {
            swap;
        }
    }

    배열 전체 출력;
}
뒤쪽은 이미 정렬되었기 때문에 안쪽 반복문은 n - 1 - i까지만 비교합니다.

4. 정답 C 코드

#include <stdio.h>

int main() {
    int n;
    int arr[101];

    scanf("%d", &n);

    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }

        for (int j = 0; j < n; j++) {
            printf("%d ", arr[j]);
        }
        printf("\n");
    }

    return 0;
}

6. 코드 자세한 설명

6.1 배열 입력

scanf("%d", &n);
for (int i = 0; i < n; i++) {
    scanf("%d", &arr[i]);
}

정렬할 숫자의 개수와 숫자들을 배열에 저장합니다.

6.2 바깥 반복문

for (int i = 0; i < n - 1; i++)

버블 정렬은 최대 n-1단계가 필요합니다. 한 단계가 끝날 때마다 뒤쪽에 정렬된 값이 하나씩 생깁니다.

6.3 안쪽 반복문

for (int j = 0; j < n - 1 - i; j++)

인접한 두 원소 arr[j]와 arr[j+1]을 비교합니다. 이미 정렬된 뒤쪽은 비교할 필요가 없기 때문에 n-1-i까지만 비교합니다.

6.4 교환

if (arr[j] > arr[j + 1]) {
    int temp = arr[j];
    arr[j] = arr[j + 1];
    arr[j + 1] = temp;
}

오름차순 정렬이므로 왼쪽 값이 오른쪽 값보다 크면 두 값을 바꿉니다.

6.5 단계별 출력

for (int j = 0; j < n; j++) {
    printf("%d ", arr[j]);
}
printf("\n");

정올 1157번의 핵심입니다. 정렬이 모두 끝난 후가 아니라, 바깥 반복문 한 단계가 끝날 때마다 출력합니다.

7. 시간복잡도

항목설명
비교 횟수(n-1) + (n-2) + ... + 1
시간복잡도O(N²)
메모리복잡도O(N)
버블 정렬은 구현은 쉽지만, 큰 데이터에는 느립니다. 이 문제는 버블 정렬의 동작 과정을 확인하는 교육용 문제입니다.

8. 핵심 정리

  1. 버블 정렬은 인접한 두 숫자를 비교합니다.
  2. 왼쪽 숫자가 더 크면 두 숫자를 교환합니다.
  3. 한 단계가 끝나면 가장 큰 값이 맨 뒤로 이동합니다.
  4. 뒤쪽은 이미 정렬되었으므로 다시 비교하지 않습니다.
  5. 정올 1157번은 각 단계가 끝날 때마다 배열을 출력해야 합니다.
  6. 시간복잡도는 O(N²)입니다.

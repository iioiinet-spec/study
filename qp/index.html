패턴 01
숫자
숫자 오름차순
return x->value - y->value;
작은 값이 앞으로 가장 기본 패턴
패턴 02
숫자
숫자 내림차순
return y->value - x->value;
큰 값이 앞으로 점수·순위 문제
패턴 03
숫자
long long 안전 비교
if (x->value < y->value) return -1; if (x->value > y->value) return 1; return 0;
큰 수 비교용 ⚠️ 오버플로우 방지
패턴 04
좌표
x 오름차순
return p->x - q->x;
x좌표 기준 오름차순
패턴 05
좌표
x 내림차순
return q->x - p->x;
x좌표 기준 내림차순
패턴 06
좌표
x↑ y↑ ⭐
if (p->x != q->x) return p->x - q->x; return p->y - q->y;
백준 좌표 정렬 기본 가장 중요
패턴 07
좌표
x↑ y↓
if (p->x != q->x) return p->x - q->x; return q->y - p->y;
x 같으면 y 내림차순
패턴 08
좌표
x↓ y↑
if (p->x != q->x) return q->x - p->x; return p->y - q->y;
x 내림, y 같으면 오름
패턴 09
좌표
x↓ y↓
if (p->x != q->x) return q->x - p->x; return q->y - p->y;
x, y 모두 내림차순
패턴 10
문자열
사전순 오름차순 ⭐
return strcmp(x->name, y->name);
문자열은 반드시 strcmp ⚠️ 절대 빼기 사용 금지
패턴 11
문자열
사전순 내림차순
return strcmp(y->name, x->name);
x↔y 순서만 뒤집기
패턴 12
문자열
길이 오름차순
return x->len - y->len;
짧은 문자열이 앞으로
패턴 13
문자열
길이↑ + 사전순 ⭐
if (x->len != y->len) return x->len - y->len; return strcmp(x->word, y->word);
백준 1181 핵심 패턴
패턴 14
문자열
길이↓ + 사전순
if (x->len != y->len) return y->len - x->len; return strcmp(x->word, y->word);
긴 단어 먼저, 같으면 사전순
패턴 15
학생
점수↓ + 이름↑ ⭐
if (x->score != y->score) return y->score - x->score; return strcmp(x->name, y->name);
점수 높은 순 같으면 이름 사전순
패턴 16
학생
다중 조건 (국영수) ⭐
if (x->kor != y->kor) return y->kor - x->kor; if (x->eng != y->eng) return x->eng - y->eng; if (x->math != y->math) return y->math - x->math; return strcmp(x->name, y->name);
시험 필수 패턴 조건 순서 중요
패턴 17
구간
시작↑ + 끝↑
if (x->start != y->start) return x->start - y->start; return x->end - y->end;
시작 시간 기준 정렬
패턴 18
구간
끝↑ + 시작↑ ⭐
if (x->end != y->end) return x->end - y->end; return x->start - y->start;
회의실 배정 핵심 그리디 필수
패턴 19
그리디
무게↑ + 가격↓
if (x->weight != y->weight) return x->weight - y->weight; return y->price - x->price;
무게 가벼운 순 같으면 비싼 순
패턴 20
입력순
나이↑ + 입력순 유지
if (x->age != y->age) return x->age - y->age; return x->order - y->order;
같은 값이면 입력순 order 필드 필수

⚠️ 자주 틀리는 패턴

실수 01
문자열 비교 실수
❌ 틀린 코드
return x->name - y->name; // 포인터 주소 비교됨!
✅ 올바른 코드
return strcmp(x->name, y->name); // 문자 내용 비교!
실수 02
조건 순서 실수
❌ 틀린 코드
// 2순위를 먼저 쓰면 안 됨 return x->eng - y->eng;
✅ 올바른 코드
if (x->kor != y->kor) return y->kor - x->kor;
실수 03
long long 오버플로우
❌ 위험한 코드
return x->value - y->value; // 큰 수 → 오버플로우!
✅ 안전한 코드
if (x->value < y->value) return -1; if (x->value > y->value) return 1; return 0;
실수 04
오름/내림 방향 혼동
// 오름차순: x - y (작은 것이 앞) return x->v - y->v; // 내림차순: y - x (큰 것이 앞) return y->v - x->v;
오름 = x-y 내림 = y-x

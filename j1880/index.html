정올 암호풀기

치환 암호 · 대소문자 유지 · 공백 유지 · fgets 입력 처리

1. 문제 핵심

복호화 키 26글자가 주어지고, 암호문 한 줄을 원문으로 바꾸는 문제입니다.

1. 복호화 키는 26개의 소문자입니다.

2. a는 key[0], b는 key[1], ..., z는 key[25]로 바뀝니다.

3. 소문자는 소문자로 출력합니다.

4. 대문자는 같은 규칙을 적용하되 대문자로 출력합니다.

5. 공백은 그대로 출력합니다.

2. 인터렉티브 암호풀기

복호화 키와 암호문을 입력하면 치환 과정을 보여줍니다.

복호화 키

암호문

문자별 변환 결과

상세 변환표

3. 복호화 키 이해하기

예를 들어 키가 다음과 같다고 합시다.

eydbkmiqugjxlvtzpnwohracsf

그러면 알파벳은 다음처럼 바뀝니다.

key[0]은 a가 바뀔 문자, key[1]은 b가 바뀔 문자입니다.

4. 소문자 처리 방법

문자가 소문자라면 ch - 'a'로 인덱스를 구합니다.

if (ch >= 'a' && ch <= 'z') {
    result = key[ch - 'a'];
}

예를 들어 ch가 'f'라면:

'f' - 'a' = 5
result = key[5]

5. 대문자 처리 방법

대문자는 같은 규칙으로 치환하지만, 출력도 대문자로 해야 합니다.

else if (ch >= 'A' && ch <= 'Z') {
    result = key[ch - 'A'] - 'a' + 'A';
}

1. ch - 'A'로 알파벳 순서를 구합니다.

2. key[index]로 소문자 결과를 찾습니다.

3. - 'a' + 'A'를 해서 대문자로 바꿉니다.

6. 공백 처리 방법

공백은 암호 규칙을 적용하지 않고 그대로 출력합니다.

else {
    result = ch;
}

7. 입력 처리 주의점

키는 공백이 없는 문자열이므로 scanf로 받을 수 있습니다.

scanf("%s", key);

하지만 암호문에는 공백이 들어갈 수 있으므로 fgets로 받아야 합니다.

getchar();
fgets(sentence, sizeof(sentence), stdin);

scanf 다음에 fgets를 바로 쓰면 남은 엔터를 읽을 수 있으므로 getchar()로 엔터를 제거합니다.

8. 자주 하는 실수

실수 1. 암호문을 scanf("%s")로 받는 경우

공백 뒤 문장이 잘립니다.

암호문은 공백을 포함할 수 있으므로 fgets(sentence, sizeof(sentence), stdin)을 사용해야 합니다.

실수 2. 대문자를 소문자로 출력하는 경우

대문자는 대문자로 유지해야 합니다.

key는 소문자이므로 대문자 입력은 key[index] - 'a' + 'A'로 바꿔 출력해야 합니다.

실수 3. 공백을 없애는 경우

공백은 그대로 출력해야 합니다.

문장 구조를 유지해야 하므로 공백 문자는 치환하지 않고 그대로 둡니다.

9. 정답 C 코드

#include <stdio.h>

int main() {
    char key[30];
    char sentence[100];

    scanf("%s", key);
    getchar();

    fgets(sentence, sizeof(sentence), stdin);

    for (int i = 0; sentence[i] != '\0'; i++) {
        if (sentence[i] >= 'a' && sentence[i] <= 'z') {
            sentence[i] = key[sentence[i] - 'a'];
        } else if (sentence[i] >= 'A' && sentence[i] <= 'Z') {
            sentence[i] = key[sentence[i] - 'A'] - 'a' + 'A';
        }
    }

    printf("%s", sentence);

    return 0;
}

10. 핵심 정리

1. 소문자는 key[ch - 'a']로 바꿉니다.

2. 대문자는 key[ch - 'A']를 찾은 뒤 대문자로 바꿉니다.

3. 공백은 그대로 둡니다.

4. 암호문은 공백이 있으므로 fgets를 사용합니다.

5. scanf 뒤 fgets 전에는 getchar()를 사용합니다.

정올 #1809 탑

각 탑이 왼쪽으로 보낸 레이저를 어느 탑이 받는지 스택으로 빠르게 찾는 과정을 배웁니다.

1. 문제를 쉽게 이해하기

레이저 방향

각 탑은 자기 왼쪽으로 레이저를 보냅니다.

누가 받나?

왼쪽에 있는 탑 중 자기보다 높거나 같은 탑이 레이저를 받습니다.

정답

가장 가까운 수신 탑 번호를 출력하고, 없으면 0을 출력합니다.

초등학생 버전: 탑이 왼쪽으로 빛을 쏩니다. 낮은 탑은 빛을 막지 못하고, 나보다 높거나 같은 탑을 만나면 거기서 멈춥니다.

2. 직접 실행해보기

탑과 레이저

스택 상태 아래 → 위 순서

실행을 누르면 낮은 탑을 제거하고 수신 탑을 찾는 과정이 표시됩니다.

스택에는 앞으로 레이저를 받을 가능성이 있는 탑만 남깁니다.

0 0 2 2 4

4. 왜 낮은 탑을 버릴까?

  1. 현재 탑보다 낮은 탑은 현재 탑의 레이저를 받을 수 없습니다.
  2. 그 낮은 탑은 앞으로 나오는 더 오른쪽 탑의 레이저도 받기 어렵습니다.
  3. 왜냐하면 현재 탑이 그 낮은 탑보다 오른쪽에 있고 더 높기 때문입니다.
  4. 그래서 낮은 탑은 스택에서 제거합니다.
  5. 제거 후 스택 맨 위에 남은 탑이 현재 탑의 레이저를 받습니다.
  6. 스택이 비어 있으면 받을 탑이 없으므로 0입니다.

5. 예제 설명

입력
5
6 9 5 7 4

풀이
1번 탑 6 → 왼쪽 없음 → 0
2번 탑 9 → 6은 낮아서 제거 → 0
3번 탑 5 → 9가 받음 → 2
4번 탑 7 → 5는 낮아서 제거, 9가 받음 → 2
5번 탑 4 → 7이 받음 → 4

출력
0 0 2 2 4

6. 정답 C 코드

#include <stdio.h>

#define MAXN 500005

typedef struct {
    int height;
    int index;
} Tower;

int main() {
    int N;
    int h;
    Tower stack[MAXN];
    int top = 0;

    scanf("%d", &N);

    for (int i = 1; i <= N; i++) {
        scanf("%d", &h);

        while (top > 0 && stack[top].height < h) {
            top--;
        }

        if (top == 0) {
            printf("0 ");
        } else {
            printf("%d ", stack[top].index);
        }

        top++;
        stack[top].height = h;
        stack[top].index = i;
    }

    return 0;
}

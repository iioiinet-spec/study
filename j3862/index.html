정올 #3862 웜홀

웜홀을 2개씩 짝지었을 때, 소가 오른쪽으로만 걷다가 무한 루프에 빠지는 짝짓기 경우의 수를 구합니다.

1. 문제를 쉽게 이해하기

웜홀

웜홀은 2개씩 짝을 이룹니다. 한 웜홀에 들어가면 짝 웜홀로 순간이동합니다.

소의 이동

소는 항상 오른쪽으로만 걷습니다. 같은 y좌표에서 오른쪽에 있는 가장 가까운 웜홀로 갑니다.

정답

모든 짝짓기 방법 중 소가 무한히 반복 이동할 수 있는 경우의 수를 셉니다.

초등학생 버전: 웜홀은 순간이동 문입니다. 소가 오른쪽으로 걷다가 웜홀을 만나면 짝꿍 웜홀로 이동합니다. 그러다 다시 오른쪽으로 걷고 또 웜홀을 타면서 계속 빙글빙글 돌면 “무한 루프”입니다.

2. 풀이 순서

  1. 각 웜홀에서 오른쪽으로 걸었을 때 가장 먼저 만나는 웜홀을 계산합니다. 이것이 nextRight입니다.
  2. 웜홀들을 2개씩 짝지을 수 있는 모든 방법을 백트래킹으로 만듭니다.
  3. 짝짓기가 하나 완성되면 사이클이 생기는지 검사합니다.
  4. 한 번 이동은 현재 웜홀 → 짝 웜홀로 순간이동 → 오른쪽 다음 웜홀로 이동입니다.
  5. 어떤 시작점에서 N번 이동해도 밖으로 나가지 못하면 무한 루프가 있습니다.
  6. 무한 루프가 있는 짝짓기 방법의 개수를 정답으로 셉니다.

4. nextRight 계산 이해

소는 오른쪽으로만 걷기 때문에, 각 웜홀에서 같은 y좌표에 있고 x가 더 큰 웜홀 중 가장 가까운 것을 찾아 둡니다.

6. 핵심 코드 작성 방법

  1. X[], Y[] 배열에 웜홀 좌표를 저장합니다.
  2. nextRight[i]를 계산합니다. 없으면 0입니다.
  3. partner[i]는 i번 웜홀의 짝 번호입니다. 아직 없으면 0입니다.
  4. solve() 함수에서 아직 짝이 없는 첫 번째 웜홀을 찾습니다.
  5. 그 웜홀을 뒤쪽의 짝 없는 웜홀과 하나씩 짝지어 봅니다.
  6. 모든 웜홀이 짝을 이루면 hasCycle()로 무한 루프를 검사합니다.
  7. 사이클이 있으면 1을 반환하고, 없으면 0을 반환합니다.
  8. 모든 경우를 더하면 정답입니다.

7. 사이클 검사 원리

현재 위치가 pos일 때 한 번의 이동은 다음 한 줄입니다.

pos = nextRight[partner[pos]];

뜻은 다음과 같습니다.

  1. 현재 웜홀 pos에 들어갑니다.
  2. partner[pos]로 순간이동합니다.
  3. 그 위치에서 오른쪽으로 걸어서 nextRight[partner[pos]]로 갑니다.
  4. 오른쪽 다음 웜홀이 없으면 pos는 0이 되어 밖으로 나간 것입니다.
  5. N번 반복해도 0이 되지 않으면 계속 도는 것이므로 사이클입니다.

8. 정답 C 코드

#include <stdio.h>

int N;
int X[13], Y[13];
int partner[13];
int nextRight[13];

int hasCycle() {
    int start, pos, cnt;

    for (start = 1; start <= N; start++) {
        pos = start;

        for (cnt = 0; cnt < N; cnt++) {
            pos = nextRight[partner[pos]];

            if (pos == 0) break;
        }

        if (pos != 0) {
            return 1;
        }
    }

    return 0;
}

int solve() {
    int i, j;
    int first = 0;
    int total = 0;

    for (i = 1; i <= N; i++) {
        if (partner[i] == 0) {
            first = i;
            break;
        }
    }

    if (first == 0) {
        if (hasCycle()) return 1;
        return 0;
    }

    for (j = first + 1; j <= N; j++) {
        if (partner[j] == 0) {
            partner[first] = j;
            partner[j] = first;

            total += solve();

            partner[first] = 0;
            partner[j] = 0;
        }
    }

    return total;
}

int main() {
    int i, j;

    scanf("%d", &N);

    for (i = 1; i <= N; i++) {
        scanf("%d %d", &X[i], &Y[i]);
    }

    for (i = 1; i <= N; i++) {
        nextRight[i] = 0;

        for (j = 1; j <= N; j++) {
            if (Y[i] == Y[j] && X[j] > X[i]) {
                if (nextRight[i] == 0 || X[j] < X[nextRight[i]]) {
                    nextRight[i] = j;
                }
            }
        }
    }

    printf("%d\n", solve());

    return 0;
}

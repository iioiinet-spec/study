정올 1535 단어집합2

fgets · strtok · strcmp · strcpy를 이용한 단어집합 만들기

1. 문제 핵심

여러 줄의 문장을 입력받아, 지금까지 나오지 않은 단어만 단어집합에 추가하는 문제입니다.

1. 처음 단어집합은 비어 있습니다.

2. 한 줄을 공백 기준으로 단어로 나눕니다.

3. 단어집합에 없는 단어만 추가합니다.

4. 한 줄 처리가 끝날 때마다 현재 단어집합을 출력합니다.

5. 입력이 END이면 출력하지 않고 종료합니다.

2. 인터렉티브 실행기

여러 줄의 문장을 입력하고 실행하면, 각 줄마다 단어집합이 어떻게 변하는지 확인할 수 있습니다.


3. 한 줄 처리 과정 직접 보기

아래 문장 한 줄을 입력하면 단어 분리와 중복 검사를 확인할 수 있습니다.

4. 코드 작성 방법

1. 문자열 입력은 fgets 사용

문장에는 공백이 있으므로 scanf("%s")만 쓰면 한 단어만 입력됩니다.

fgets(line, sizeof(line), stdin);

2. 줄바꿈 제거

line[strcspn(line, "\n")] = '\0';

3. END 검사

if (strcmp(line, "END") == 0) {
    break;
}

4. strtok로 단어 분리

char *token = strtok(line, " ");

while (token != NULL) {
    ...
    token = strtok(NULL, " ");
}

5. 이미 있는 단어인지 확인

int exists(char word[]) {
    for (int i = 0; i < wordCount; i++) {
        if (strcmp(words[i], word) == 0) return 1;
    }
    return 0;
}

6. 새 단어 저장

if (!exists(token)) {
    strcpy(words[wordCount], token);
    wordCount++;
}

5. 왜 배열을 크게 잡는가?

입력 줄이 길거나 단어 개수가 많으면 작은 배열은 버퍼 범위를 넘을 수 있습니다.

char words[10000][105];
char line[10005];

6. 자주 하는 실수

실수 1. scanf("%s")로 문장을 입력받는 경우

공백이 있는 문장을 처리할 수 없습니다.

scanf("%s")는 공백 전까지만 읽습니다. 이 문제는 한 줄 전체를 읽어야 하므로 fgets가 적합합니다.

실수 2. END를 출력해버리는 경우

END가 나오면 바로 종료해야 합니다.

END는 단어집합에 추가하지 않고, 현재 단어집합도 출력하지 않습니다.

실수 3. 단어집합을 매 줄마다 초기화하는 경우

단어집합은 프로그램이 끝날 때까지 유지됩니다.

첫 번째 줄에서 나온 단어는 두 번째 줄에서도 이미 나온 단어로 판단해야 합니다.

실수 4. 대소문자를 같은 단어로 처리하는 경우

strcmp는 대소문자를 구분합니다.

Apple과 apple은 서로 다른 단어로 처리됩니다.

7. 정답 C 코드

#include <stdio.h>
#include <string.h>

char words[10000][105];
int wordCount = 0;

int exists(char word[]) {
    for (int i = 0; i < wordCount; i++) {
        if (strcmp(words[i], word) == 0) return 1;
    }
    return 0;
}

void printWords() {
    for (int i = 0; i < wordCount; i++) {
        if (i > 0) printf(" ");
        printf("%s", words[i]);
    }
    printf("\n");
}

int main() {
    char line[10005];

    while (fgets(line, sizeof(line), stdin) != NULL) {
        line[strcspn(line, "\n")] = '\0';

        if (strcmp(line, "END") == 0) break;

        char *token = strtok(line, " ");

        while (token != NULL) {
            if (!exists(token)) {
                strcpy(words[wordCount], token);
                wordCount++;
            }
            token = strtok(NULL, " ");
        }

        printWords();
    }
    return 0;
}

8. 핵심 정리

1. 문장 입력은 fgets를 사용합니다.

2. 줄바꿈 제거는 strcspn을 사용합니다.

3. 단어 분리는 strtok를 사용합니다.

4. 단어 비교는 strcmp를 사용합니다.

5. 새 단어 저장은 strcpy를 사용합니다.

6. 단어집합은 계속 유지됩니다.

7. END가 나오면 출력하지 않고 종료합니다.

숫자의 개수

세 자연수 A, B, C를 곱한 결과에서 숫자 0부터 9까지 각각 몇 번 나오는지 세는 문제입니다. 핵심은 나머지 연산 % 10과 배열 count[10]입니다.

1. 문제 요약

세 자연수 A, B, C를 입력받아 A × B × C 결과에 포함된 숫자의 개수를 출력합니다.

입력 예시

150
266
427

출력 예시

3
1
0
2
0
0
0
2
0
0

2. 핵심 아이디어

  1. 세 수를 곱해서 결과값을 만듭니다.
  2. 결과값의 마지막 자리 숫자를 % 10으로 꺼냅니다.
  3. 꺼낸 숫자의 개수를 count[숫자]에 저장합니다.
  4. 마지막 자리를 / 10으로 제거합니다.

3. 배열의 의미

count[10]은 숫자 0부터 9까지의 개수를 저장하는 배열입니다.

배열의미
count[0]숫자 0의 개수
count[1]숫자 1의 개수
count[2]숫자 2의 개수
......
count[9]숫자 9의 개수

4. 핵심 연산

마지막 자리 숫자 = result % 10
마지막 자리 제거 = result / 10

예를 들어 result가 17037300이라면 다음처럼 오른쪽 숫자부터 하나씩 확인합니다.

17037300 % 10 = 0
1703730  % 10 = 0
170373   % 10 = 3
17037    % 10 = 7
...

5. 인터랙티브 계산기

세 수를 바꾸면 곱셈 결과, 자리 숫자 분해, 숫자별 개수가 자동으로 바뀝니다.

A × B × C
17037300

6. 결과 숫자 분해

곱셈 결과에 들어 있는 숫자들을 한 자리씩 보여줍니다.

숫자 개수 막대그래프

8. while문 동작 과정

오른쪽 끝 숫자를 꺼내고, 그 숫자를 제거하는 과정을 반복합니다.

9. 코드 작성 순서

  1. A, B, C를 입력받습니다.
  2. result = A × B × C를 계산합니다.
  3. count[10] 배열을 모두 0으로 준비합니다.
  4. result가 0보다 큰 동안 마지막 자리 숫자를 꺼냅니다.
  5. count[digit]을 1 증가시킵니다.
  6. 0부터 9까지 count 값을 한 줄씩 출력합니다.

10. 정답 C 코드

#include <stdio.h>

int main() {
    int A, B, C;
    int result;
    int count[10] = {0};

    scanf("%d", &A);
    scanf("%d", &B);
    scanf("%d", &C);

    result = A * B * C;

    while (result > 0) {
        int digit = result % 10;
        count[digit]++;
        result = result / 10;
    }

    for (int i = 0; i < 10; i++) {
        printf("%d\n", count[i]);
    }

    return 0;
}
코드가 복사되었습니다.

11. 코드 자세한 설명

1. 배열 준비

int count[10] = {0};

숫자 0부터 9까지의 개수를 저장하기 위해 크기 10짜리 배열을 만듭니다.

2. 마지막 자리 꺼내기

int digit = result % 10;

어떤 수를 10으로 나눈 나머지는 항상 마지막 자리 숫자입니다.

3. 개수 증가

count[digit]++;

digit이 7이면 count[7]이 1 증가합니다. 즉, 7이 하나 더 발견된 것입니다.

4. 마지막 자리 제거

result = result / 10;

정수 나눗셈이므로 마지막 자리가 사라집니다. 예를 들어 17037300 / 10 = 1703730입니다.

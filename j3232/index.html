정올 #3232 두 박스

두 직사각형 박스가 넓게 겹치는지, 선으로 만나는지, 점으로 만나는지, 아예 안 만나는지 직접 확인해 봅시다.

1. 문제를 쉽게 이해하기

박스 2개가 있습니다. 각 박스는 왼쪽 아래 점과 오른쪽 위 점으로 표현합니다.

첫 번째 박스: (x1, y1), (x2, y2)
두 번째 박스: (x3, y3), (x4, y4)

색종이 2장을 생각하면 쉽습니다. 색종이가 넓게 겹치면 FACE, 테두리만 닿으면 LINE, 꼭짓점만 닿으면 POINT, 떨어져 있으면 NULL입니다.

2. 네 가지 결과

FACE 넓이가 있게 겹침 가로도 겹치고, 세로도 겹칩니다.
LINE 선분으로 만남 한쪽 방향은 딱 붙고, 다른 방향은 겹칩니다.
POINT 한 점에서 만남 꼭짓점 하나만 서로 닿습니다.
NULL 만나지 않음 가로 또는 세로 방향으로 떨어져 있습니다.

3. 직접 좌표를 바꿔보기

좌표 입력

좌표 범위는 보기 쉽게 -10부터 10 사이를 추천합니다.

박스 A

박스 B

판정 결과 FACE

가로와 세로가 모두 겹칩니다.

4. 핵심 공식

두 박스가 얼마나 겹치는지 가로 길이와 세로 길이를 계산합니다.

w = min(x2, x4) - max(x1, x3)
h = min(y2, y4) - max(y1, y3)
  1. w > 0이고 h > 0이면 넓이가 있으므로 FACE입니다.
  2. w == 0이고 h > 0이거나, w > 0이고 h == 0이면 선으로 만나므로 LINE입니다.
  3. w == 0이고 h == 0이면 꼭짓점 하나만 만나므로 POINT입니다.
  4. 그 외에는 떨어져 있으므로 NULL입니다.

5. 초등학생도 이해하는 설명

  1. 박스 A와 박스 B를 색종이 2장이라고 생각합니다.
  2. 색종이 2장이 겹쳐서 색이 진하게 보이는 부분이 있으면 FACE입니다.
  3. 색종이의 변끼리 딱 붙어 있으면 LINE입니다.
  4. 색종이의 모서리 한 점만 콕 닿으면 POINT입니다.
  5. 서로 멀리 떨어져 있으면 NULL입니다.

6. 정답 C 코드

#include <stdio.h>

long long min(long long a, long long b) {
    if (a < b) return a;
    return b;
}

long long max(long long a, long long b) {
    if (a > b) return a;
    return b;
}

int main() {
    long long x1, y1, x2, y2;
    long long x3, y3, x4, y4;
    long long w, h;

    scanf("%lld %lld %lld %lld", &x1, &y1, &x2, &y2);
    scanf("%lld %lld %lld %lld", &x3, &y3, &x4, &y4);

    w = min(x2, x4) - max(x1, x3);
    h = min(y2, y4) - max(y1, y3);

    if (w > 0 && h > 0) {
        printf("FACE\n");
    }
    else if ((w == 0 && h > 0) || (w > 0 && h == 0)) {
        printf("LINE\n");
    }
    else if (w == 0 && h == 0) {
        printf("POINT\n");
    }
    else {
        printf("NULL\n");
    }

    return 0;
}

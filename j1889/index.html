정올 #1889 N Queen

N×N 체스판에 N개의 퀸을 서로 공격하지 않게 놓는 방법의 개수를 백트래킹으로 찾아봅니다.

1. 문제를 쉽게 이해하기

퀸의 공격 방향

퀸은 가로, 세로, 대각선으로 움직일 수 있습니다.

목표

N개의 퀸이 서로 공격하지 않도록 놓는 모든 방법의 개수를 구합니다.

핵심 전략

한 행에 퀸을 하나씩 놓고, 불가능하면 되돌아가는 백트래킹을 사용합니다.

초등학생 버전: 퀸은 아주 강한 말입니다. 같은 줄, 같은 세로줄, 대각선에 친구 퀸이 있으면 공격하게 됩니다. 그래서 “여기 놓아도 안전한가?”를 확인하면서 한 줄씩 놓아봅니다.

2. 직접 놓아보고 자동 탐색 보기

체스판을 클릭하면 직접 퀸을 놓을 수 있습니다. 빨간 칸은 공격받는 칸입니다.

4. DFS + 백트래킹 과정

자동 DFS 실행을 누르면 퀸을 어디에 놓아보고, 어디에서 충돌하거나 되돌아가는지 확인할 수 있습니다.

순서 행 열 현재 배치 상태 설명

5. 왜 대각선을 이렇게 확인할까?

같은 열

col[c]

같은 열 c에 이미 퀸이 있으면 놓을 수 없습니다.

오른쪽 아래 대각선

diag1[row + c]

같은 방향 대각선은 row + c 값이 같습니다.

왼쪽 아래 대각선

diag2[row - c + N - 1]

row - c는 음수가 될 수 있어서 N-1을 더합니다.

쉽게 말하면, 열 번호표와 대각선 번호표를 만들어서 “이미 퀸이 지나가는 길인가?”를 빠르게 확인하는 것입니다.

6. 핵심 알고리즘 설명

  1. 0행부터 시작해서 한 행에 퀸을 하나씩 놓습니다.
  2. 현재 행에서 0열부터 N-1열까지 차례대로 놓아봅니다.
  3. 같은 열, 같은 대각선에 퀸이 없으면 안전합니다.
  4. 안전하면 퀸을 놓고 다음 행으로 내려갑니다.
  5. 끝까지 N개를 모두 놓으면 정답 개수를 1 증가시킵니다.
  6. 다른 경우도 찾아야 하므로 방금 놓은 퀸을 다시 지웁니다.
  7. 이렇게 놓기와 지우기를 반복하는 것이 백트래킹입니다.

7. 정답 C 코드

#include <stdio.h>

int N;
int answer = 0;

int col[14];
int diag1[30];
int diag2[30];

void dfs(int row) {
    int c;

    if (row == N) {
        answer++;
        return;
    }

    for (c = 0; c < N; c++) {
        if (col[c] == 0 && diag1[row + c] == 0 && diag2[row - c + N - 1] == 0) {
            col[c] = 1;
            diag1[row + c] = 1;
            diag2[row - c + N - 1] = 1;

            dfs(row + 1);

            col[c] = 0;
            diag1[row + c] = 0;
            diag2[row - c + N - 1] = 0;
        }
    }
}

int main() {
    scanf("%d", &N);

    dfs(0);

    printf("%d\n", answer);

    return 0;
}

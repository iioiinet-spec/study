정올 #3297 구간의 합(PURQ) 1

세그먼트 트리로 한 위치 값 변경과 구간 합 질의를 빠르게 처리하는 과정을 시각적으로 배웁니다.

1. 세그먼트 트리란?

구간을 반으로 나눔

전체 구간을 계속 반으로 나누어 각 구간의 합을 저장합니다.

[1~8]
→ [1~4], [5~8]

Point Update

한 위치 값이 바뀌면 그 위치를 포함하는 구간 합만 다시 계산합니다.

1 a b

Range Query

질의 구간에 완전히 포함되는 노드만 골라 더합니다.

2 left right

초등학생 버전: 숫자 카드를 큰 상자, 중간 상자, 작은 상자로 나누어 각 상자의 합을 미리 적어 둡니다. 필요한 구간이 나오면 맞는 상자만 골라 더합니다.

4. 세그먼트 트리 함수 이해

  1. init(node, start, end)는 node가 담당하는 start~end 구간의 합을 만듭니다.
  2. start == end이면 실제 배열 한 칸이므로 tree[node] = arr[start]입니다.
  3. 아니면 왼쪽 자식과 오른쪽 자식의 합을 더해서 현재 노드에 저장합니다.
  4. update는 바꾸려는 idx가 포함된 쪽으로만 내려갑니다.
  5. 리프 노드에서 값을 바꾼 뒤, 돌아오면서 부모 구간 합을 다시 계산합니다.
  6. query는 구간이 겹치지 않으면 0을 반환합니다.
  7. 구간이 완전히 포함되면 tree[node]를 바로 반환합니다.
  8. 일부만 겹치면 왼쪽과 오른쪽으로 나누어 다시 확인합니다.

5. 핵심 조건 3가지

겹치지 않음

right < start || end < left

질의 구간과 현재 노드 구간이 안 겹치면 0을 반환합니다.

완전히 포함

left <= start && end <= right

현재 노드 구간이 질의 구간 안에 전부 들어오면 tree[node]를 바로 씁니다.

일부 겹침

왼쪽 query + 오른쪽 query

반으로 나누어 두 자식을 확인합니다.

6. 정답 C 코드

#include <stdio.h>

#define MAX 1000001

int N, M;
long long arr[MAX];
long long tree[MAX * 4];

long long init(int node, int start, int end) {
    if (start == end) {
        tree[node] = arr[start];
        return tree[node];
    }

    int mid = (start + end) / 2;

    tree[node] = init(node * 2, start, mid)
               + init(node * 2 + 1, mid + 1, end);

    return tree[node];
}

void update(int node, int start, int end, int idx, long long value) {
    if (idx < start || idx > end) {
        return;
    }

    if (start == end) {
        arr[idx] = value;
        tree[node] = value;
        return;
    }

    int mid = (start + end) / 2;

    update(node * 2, start, mid, idx, value);
    update(node * 2 + 1, mid + 1, end, idx, value);

    tree[node] = tree[node * 2] + tree[node * 2 + 1];
}

long long query(int node, int start, int end, int left, int right) {
    if (right < start || end < left) {
        return 0;
    }

    if (left <= start && end <= right) {
        return tree[node];
    }

    int mid = (start + end) / 2;

    return query(node * 2, start, mid, left, right)
         + query(node * 2 + 1, mid + 1, end, left, right);
}

int main() {
    int i;
    int cmd, a;
    long long b;

    scanf("%d", &N);

    for (i = 1; i <= N; i++) {
        scanf("%lld", &arr[i]);
    }

    init(1, 1, N);

    scanf("%d", &M);

    for (i = 0; i < M; i++) {
        scanf("%d %d %lld", &cmd, &a, &b);

        if (cmd == 1) {
            update(1, 1, N, a, b);
        } else {
            printf("%lld\n", query(1, 1, N, a, (int)b));
        }
    }

    return 0;
}

정올 #2577 회전초밥(고)

원형 초밥 벨트에서 연속 k개를 먹을 때, 쿠폰 초밥까지 포함해 먹을 수 있는 최대 종류 수를 슬라이딩 윈도우로 구합니다.

1. 문제를 쉽게 이해하기

회전 벨트

초밥 접시가 원형으로 놓여 있습니다. 끝까지 가면 다시 처음으로 돌아옵니다.

연속 k개 선택

어떤 위치에서 시작해서 연속된 k개의 접시를 먹습니다.

쿠폰 초밥

선택한 k개 안에 쿠폰 초밥이 없으면 쿠폰으로 한 종류를 더 먹을 수 있습니다.

초등학생 버전: 초밥 기차에서 연속으로 k접시를 골라 먹습니다. 그런데 쿠폰 초밥이 내가 고른 접시 안에 없으면 공짜로 하나 더 먹을 수 있습니다. 그래서 “서로 다른 초밥 종류가 최대 몇 개인가?”를 찾습니다.

2. 직접 입력하고 창 이동 보기

원형 초밥 벨트

현재 종류 수 4 쿠폰 +1

입력 형식: 첫 줄 N d k c, 다음 줄부터 초밥 번호 N개입니다.

정답 최대 종류 수

5

현재 시작 위치

0

현재 창의 종류 수

4

쿠폰 포함 결과

5

3. 현재 먹는 k개 접시

슬라이딩 실행을 누르면 현재 창의 종류 수와 쿠폰 처리를 확인할 수 있습니다.

4. 슬라이딩 윈도우 과정

한 칸 이동할 때마다 처음부터 다시 세지 않고, 맨 앞 초밥 1개 제거와 새 초밥 1개 추가만 합니다.

시작 위치 제거 추가 창 안 종류 쿠폰 총 종류 최댓값

5. 핵심 알고리즘 설명

  1. 처음 0번 접시부터 k개를 선택합니다.
  2. cnt[초밥번호] 배열에 현재 창 안에 각 초밥이 몇 개 있는지 저장합니다.
  3. 새로운 초밥 종류가 들어오면 kind를 1 증가시킵니다.
  4. 창을 한 칸 옮길 때는 왼쪽 끝 초밥을 제거하고 오른쪽 새 초밥을 추가합니다.
  5. 벨트가 원형이므로 인덱스가 N을 넘어가면 % N으로 처음으로 돌아갑니다.
  6. 쿠폰 초밥 c가 현재 창 안에 없으면 kind + 1을 계산합니다.
  7. 모든 시작 위치를 확인하면서 가장 큰 값을 정답으로 저장합니다.

6. 정답 C 코드

#include <stdio.h>

int sushi[3000001];
int cnt[3001];

int main() {
    int N, d, k, c;
    int i;
    int kind = 0;
    int answer = 0;

    scanf("%d %d %d %d", &N, &d, &k, &c);

    for (i = 0; i < N; i++) {
        scanf("%d", &sushi[i]);
    }

    // 처음 k개 접시를 선택
    for (i = 0; i < k; i++) {
        if (cnt[sushi[i]] == 0) {
            kind++;
        }
        cnt[sushi[i]]++;
    }

    // 쿠폰 초밥 포함 여부 확인
    if (cnt[c] == 0) {
        answer = kind + 1;
    } else {
        answer = kind;
    }

    // 시작 위치를 1부터 N-1까지 이동
    for (i = 1; i < N; i++) {
        int remove = sushi[i - 1];
        int add = sushi[(i + k - 1) % N];

        // 맨 앞 초밥 제거
        cnt[remove]--;
        if (cnt[remove] == 0) {
            kind--;
        }

        // 새 초밥 추가
        if (cnt[add] == 0) {
            kind++;
        }
        cnt[add]++;

        // 쿠폰 초밥 계산
        if (cnt[c] == 0) {
            if (kind + 1 > answer) {
                answer = kind + 1;
            }
        } else {
            if (kind > answer) {
                answer = kind;
            }
        }
    }

    printf("%d\n", answer);

    return 0;
}

7. 초등학생도 이해하는 한 줄 정리

초밥 창문을 한 칸씩 옮기면서, 나간 접시 하나 빼고 새로 들어온 접시 하나만 더해 계산하는 방법입니다. 그래서 빠릅니다.

브론즈 1. 다항식 출력하기

순차 자료구조 배열로 다항식을 저장하고, 높은 차수부터 출력하는 과정을 직접 확인해 봅니다.

1. 핵심 개념

다항식을 배열에 저장할 때 가장 쉬운 방법은 배열의 인덱스를 차수로 사용하는 것입니다.

coef[0] = 상수항coef[1] = x의 계수coef[i] = x^i의 계수

예를 들어 7 5 0 2 3이 입력되면 다음과 같습니다.

인덱스 i01234
계수 coef[i]75023
항75x0x^22x^33x^4

출력할 때는 높은 차수부터 출력해야 하므로 N부터 0까지 거꾸로 반복합니다.

6. 정답 코드 C

#include <stdio.h>

int main()
{
    int N;
    int coef[101];

    scanf("%d", &N);

    for(int i = 0; i <= N; i++)
        scanf("%d", &coef[i]);

    int first = 1;

    for(int i = N; i >= 0; i--)
    {
        if(coef[i] == 0)
            continue;

        if(!first)
            printf(" + ");

        if(i == 0)
            printf("%d", coef[i]);
        else if(i == 1)
            printf("%dx", coef[i]);
        else
            printf("%dx^%d", coef[i], i);

        first = 0;
    }

    return 0;
}

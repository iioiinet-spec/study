순차 자료구조로 다항식 Polynomial 저장하기

배열을 이용해 다항식을 저장하는 두 가지 방법을 직접 실습해 봅니다.

1. 핵심 개념

다항식은 여러 개의 항으로 이루어진 식입니다.

P(x) = 3x⁴ + 2x³ + 5x + 7

각 항은 계수와 차수로 나눌 수 있습니다.

항계수차수
3x⁴34
2x³23
5x51
770

2. 방법 1: 계수 배열 방식

배열의 인덱스를 차수로 사용합니다.
coef[i] = xᶦ의 계수

예를 들어 P(x) = 3x⁴ + 2x³ + 5x + 7 은 다음처럼 저장합니다.

4. 방법 2: 구조체 배열 방식

0이 아닌 항만 저장합니다.
각 항을 {계수, 차수} 형태로 저장합니다.

예를 들어 P(x) = 5x¹⁰⁰⁰ + 3 은 실제 항이 2개뿐입니다.

typedef struct {
    int coef;  // 계수
    int exp;   // 차수
} Term;

Term poly[2] = {
    {5, 1000},
    {3, 0}
};

7. 두 방식 비교

방식저장 방법장점단점
계수 배열인덱스 = 차수구현이 쉽고 덧셈이 간단함차수가 크면 0이 많이 저장됨
구조체 배열{계수, 차수} 저장0인 항을 저장하지 않아 공간 절약덧셈과 곱셈 구현이 더 복잡함
P(x) = 5x¹⁰⁰⁰ + 3 처럼 차수는 크지만 항이 적은 경우에는 구조체 배열 방식이 더 효율적입니다.

8. 확인 문제

문제 1. 배열 coef = {4, 0, 2, 1} 이면 다항식은?

1x³ + 2x² + 4, 즉 x³ + 2x² + 4

문제 2. P(x) = 6x⁵ + 2x + 9 를 계수 배열로 저장하면?

coef[0]=9, coef[1]=2, coef[2]=0, coef[3]=0, coef[4]=0, coef[5]=6

문제 3. P(x) = 7x¹⁰⁰ + 3x² 처럼 항이 적고 차수가 큰 경우 어떤 방식이 좋은가?

구조체 배열 방식이 좋습니다. 0인 항을 많이 저장하지 않아도 되기 때문입니다.

9. C언어 예제 코드

#include <stdio.h>

typedef struct {
    int coef;
    int exp;
} Term;

int main(void) {
    int coef[5] = {7, 5, 0, 2, 3};

    printf("계수 배열 방식:\\n");
    for(int i = 4; i >= 0; i--) {
        if(coef[i] != 0) {
            printf("%dx^%d ", coef[i], i);
        }
    }

    Term poly[2] = {
        {5, 1000},
        {3, 0}
    };

    printf("\\n\\n구조체 배열 방식:\\n");
    for(int i = 0; i < 2; i++) {
        printf("%dx^%d ", poly[i].coef, poly[i].exp);
    }

    return 0;
}

Horner 알고리즘 인터렉티브 강의자료

다항식 P(x)를 빠르고 안전하게 계산하는 방법을 단계별로 배웁니다.

1. Horner 알고리즘이란?

Horner 알고리즘은 다항식 값을 계산할 때 거듭제곱을 직접 만들지 않고, 곱하기와 더하기만 반복하는 방법입니다.

P(x) = a₀ + a₁x + a₂x² + ... + aₙxⁿ

이 식을 다음과 같이 바꿉니다.

P(x) = (((aₙx + aₙ₋₁)x + aₙ₋₂) ... x + a₀)

핵심 규칙은 하나입니다. result = result × x + 다음 계수

2. 예제로 이해하기

다음 다항식을 보겠습니다.

P(x) = 3x⁴ + 2x³ + 0x² + 5x + 7

Horner 형태로 바꾸면 다음과 같습니다.

P(x) = ((((3x + 2)x + 0)x + 5)x + 7)

x = 2일 때 계산 과정은 다음과 같습니다.

단계계산result
시작result = 33
13 × 2 + 28
28 × 2 + 016
316 × 2 + 537
437 × 2 + 781
따라서 P(2) = 81

직접 입력해서 계산하기

계수는 낮은 차수부터 입력합니다.

예: 7 5 0 2 3 → 3x^4 + 2x^3 + 5x + 7

일반 계산과 Horner 계산 비교

구분 일반 계산 Horner 계산
계산 방식 aᵢ × xⁱ를 각각 계산 result = result × x + aᵢ 반복
거듭제곱 x², x³, x⁴ 등을 만들어야 함 직접 만들지 않음
필요 변수 result, power 등이 필요 result 하나로 가능
계산 방향 낮은 차수부터 가능 높은 차수부터 계산
시간복잡도 O(N), 잘못 구현하면 O(N²) O(N)

연산 횟수 체험

거듭제곱을 매번 반복문으로 다시 구하는 잘못된 방식과 Horner 방식을 비교합니다.

나머지 정리와 연결

Horner 알고리즘은 조립제법과도 연결됩니다.

P(x)를 x - k로 나눈 나머지 = P(k)

즉, P(2)를 Horner로 구하면 P(x)를 x-2로 나누었을 때의 나머지도 함께 구한 것입니다.

C 코드 1. 기본 Horner 알고리즘

#include <stdio.h>

int main(void)
{
    int N;
    int coef[101];
    int x;

    scanf("%d", &N);

    for(int i = 0; i <= N; i++)
    {
        scanf("%d", &coef[i]);
    }

    scanf("%d", &x);

    int result = coef[N];

    for(int i = N - 1; i >= 0; i--)
    {
        result = result * x + coef[i];
    }

    printf("%d", result);

    return 0;
}

C 코드 2. MOD가 있는 Horner 알고리즘

#include <stdio.h>

#define MOD 1000000007

long long coef[1000001];

int main(void)
{
    int N;
    long long x;

    scanf("%d", &N);

    for(int i = 0; i <= N; i++)
    {
        scanf("%lld", &coef[i]);
        coef[i] %= MOD;
    }

    scanf("%lld", &x);
    x %= MOD;

    long long result = coef[N];

    for(int i = N - 1; i >= 0; i--)
    {
        result = (result * x + coef[i]) % MOD;
    }

    printf("%lld", result);

    return 0;
}

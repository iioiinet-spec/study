다항식 문제 8선

배열로 다항식을 저장하고 출력·덧셈·계산·곱셈·Horner 알고리즘까지 연습합니다.

문제 선택
문제 1

다항식 출력하기

문제

차수 N인 다항식의 계수가 입력된다. 계수 배열은 낮은 차수부터 입력된다.

coef[0] = x^0의 계수
coef[1] = x^1의 계수
coef[2] = x^2의 계수
...
coef[N] = x^N의 계수

계수가 0이 아닌 항만 높은 차수부터 출력하시오.

입력

첫 번째 줄에 다항식의 차수 N이 입력된다.

두 번째 줄에 N+1개의 정수가 입력된다.

출력

다항식을 높은 차수부터 출력한다.

입력 예제

4
7 5 0 2 3

출력 예제

3x^4 + 2x^3 + 5x + 7

핵심 생각

입력은 낮은 차수부터 들어오지만 출력은 높은 차수부터 해야 하므로, 반복문을 i=N부터 0까지 감소시키면 된다.

문제 2

두 다항식의 덧셈

문제

두 다항식 A(x), B(x)가 계수 배열 형태로 입력된다. 두 다항식을 더한 결과 C(x)를 출력하시오.

입력

첫 번째 줄에 두 다항식의 차수 N이 입력된다.

두 번째 줄에 A(x)의 계수 N+1개가 낮은 차수부터 입력된다.

세 번째 줄에 B(x)의 계수 N+1개가 낮은 차수부터 입력된다.

출력

두 다항식을 더한 결과를 높은 차수부터 출력한다.

입력 예제

4
7 5 0 2 3
3 0 4 0 1

출력 예제

4x^4 + 2x^3 + 4x^2 + 5x + 10

핵심 생각

같은 차수끼리 더한다. 즉 C[i] = A[i] + B[i]이다.

문제 3

다항식 값 계산하기

문제

다항식 P(x)의 계수와 정수 x가 주어진다. P(x)의 값을 계산하시오.

다항식은 계수 배열 형태로 입력된다.

입력

첫 번째 줄에 다항식의 차수 N이 입력된다.

두 번째 줄에 N+1개의 계수가 낮은 차수부터 입력된다.

세 번째 줄에 정수 x가 입력된다.

출력

P(x)의 값을 출력한다.

입력 예제

4
7 5 0 2 3
2

출력 예제

73

주의

예제의 다항식은 3x^4 + 2x^3 + 5x + 7이고, x=2일 때 값은 48 + 16 + 10 + 7 = 81이다.

따라서 수학적으로는 81이 맞고, 위 출력 예제 73은 오타 가능성이 높다.

브론즈 1

다항식 출력하기

문제

컴퓨터는 다항식을 배열을 이용하여 저장할 수 있다. 배열의 인덱스는 차수, 배열의 값은 계수를 의미한다.

coef[0] = 7
coef[1] = 5
coef[2] = 0
coef[3] = 2
coef[4] = 3

이라면 P(x)=3x^4+2x^3+5x+7을 의미한다.

차수 N과 각 차수의 계수가 입력될 때 다항식을 높은 차수부터 출력하는 프로그램을 작성하시오.

조건

1. 계수가 0인 항은 출력하지 않는다.

2. 계수가 1이어도 그대로 1을 출력한다.

3. + 기호는 항 사이에만 출력한다.

입력

첫 번째 줄에 다항식의 최고차수 N이 입력된다. 1 ≤ N ≤ 100

두 번째 줄에는 coef[0] coef[1] ... coef[N]이 공백으로 구분되어 입력된다.

-100 ≤ 계수 ≤ 100

출력

다항식을 높은 차수부터 출력한다.

입출력 예제

입력 예제 1
4
7 5 0 2 3

출력 예제 1
3x^4 + 2x^3 + 5x + 7

입력 예제 2
5
1 0 0 0 0 6

출력 예제 2
6x^5 + 1

입력 예제 3
3
0 0 0 8

출력 예제 3
8x^3

핵심 생각

first 변수를 사용하면 첫 항 앞에는 +를 출력하지 않고, 두 번째 항부터 + 를 붙일 수 있다.

실버 1

두 다항식 덧셈

문제

두 다항식 A(x), B(x)가 계수 배열 형태로 주어진다.

배열의 인덱스는 차수이고, 배열의 값은 계수이다.

두 다항식을 더한 결과 C(x)를 높은 차수부터 출력하시오.

입력

첫 번째 줄에 최고차수 N이 입력된다.

두 번째 줄에 A(x)의 계수 N+1개가 낮은 차수부터 입력된다.

세 번째 줄에 B(x)의 계수 N+1개가 낮은 차수부터 입력된다.

출력

A(x) + B(x)의 결과를 높은 차수부터 출력한다.

입력 예제

4
7 5 0 2 3
3 0 4 0 1

출력 예제

4x^4 + 2x^3 + 4x^2 + 5x + 10

핵심 생각

출력 함수는 브론즈 문제와 같이 재사용할 수 있다. 덧셈은 배열의 같은 인덱스끼리 더하면 된다.

실버 2

다항식 값 계산하기

문제

다항식 P(x)의 계수와 정수 x가 주어진다.

다항식은 계수 배열 형태로 저장되어 있다.

coef[0] = x^0의 계수
coef[1] = x^1의 계수
coef[2] = x^2의 계수
...
coef[N] = x^N의 계수

P(x)의 값을 계산하여 출력하시오.

입력

첫 번째 줄에 다항식의 최고차수 N이 입력된다.

두 번째 줄에 계수 N+1개가 낮은 차수부터 입력된다.

세 번째 줄에 정수 x가 입력된다.

출력

P(x)의 값을 출력한다.

입력 예제

4
7 5 0 2 3
2

출력 예제

81

계산 과정

3x^4 + 2x^3 + 5x + 7에 x=2를 넣으면 3×16 + 2×8 + 5×2 + 7 = 81이다.

골드 1

두 다항식 곱셈

문제

두 다항식 A(x), B(x)가 계수 배열 형태로 주어진다.

배열의 인덱스는 차수이고, 배열의 값은 계수이다.

두 다항식을 곱한 결과 C(x)를 높은 차수부터 출력하시오.

입력

첫 번째 줄에 다항식 A(x)의 최고차수 N이 입력된다.

두 번째 줄에 A(x)의 계수 N+1개가 낮은 차수부터 입력된다.

세 번째 줄에 다항식 B(x)의 최고차수 M이 입력된다.

네 번째 줄에 B(x)의 계수 M+1개가 낮은 차수부터 입력된다.

출력

A(x) × B(x)의 결과를 높은 차수부터 출력한다.

입력 예제

2
1 2 3
1
4 5

출력 예제

15x^3 + 22x^2 + 13x + 4

핵심 생각

A[i]x^i × B[j]x^j = A[i]B[j]x^(i+j)이므로 C[i+j] += A[i] * B[j]를 수행한다.

플래티넘 1

다항식 빠른 계산과 나머지

문제

다항식 P(x)가 계수 배열 형태로 주어진다.

P(x) = a0 + a1x + a2x^2 + ... + anx^n

정수 K가 주어질 때, P(K)를 계산하시오.

단, 값이 매우 커질 수 있으므로 결과를 1,000,000,007로 나눈 나머지를 출력한다.

일반적인 방식으로 ai × K^i를 매번 계산하면 시간이 오래 걸린다. 따라서 Horner 알고리즘을 이용하여 빠르게 계산해야 한다.

입력

첫 번째 줄에 다항식의 최고차수 N이 입력된다.

두 번째 줄에 N+1개의 계수 a0, a1, ..., an이 낮은 차수부터 입력된다.

세 번째 줄에 정수 K가 입력된다.

출력

P(K)를 1,000,000,007로 나눈 나머지를 출력한다.

제한

1 ≤ N ≤ 1,000,000

0 ≤ ai ≤ 1,000,000,000

0 ≤ K ≤ 1,000,000,000

입력 예제

4
7 5 0 2 3
2

출력 예제

81

Horner 형태

3x^4 + 2x^3 + 0x^2 + 5x + 7은 다음처럼 바꿀 수 있다.

((((3)x + 2)x + 0)x + 5)x + 7

높은 차수 계수부터 차례대로 result = result * K + coef[i]를 반복하면 빠르게 계산할 수 있다.

📐 10진수 → 2진수 (나눗셈법)
10진수를 2로 반복해서 나누어 나머지를 거꾸로 읽으면 2진수가 됩니다.
📐 2진수 → 10진수 (자릿값법)
각 자리에 2의 거듭제곱 가중치를 곱하여 합산합니다.
⚡ 2진수 ↔ 16진수 (4비트 묶음법)
2진수 4비트 = 16진수 1자리. 4비트씩 묶어서 16진수로 변환하면 됩니다.

// Complement — 컴퓨터에서 음수를 표현하는 방법
컴퓨터는 뺄셈을 덧셈으로 처리하기 위해 보수(Complement)를 사용합니다. 3가지 방법: 부호-크기(Sign-Magnitude), 1의 보수(1's Complement), 2의 보수(2's Complement).

// Floating Point — 실수를 컴퓨터에서 표현하는 방법
실수는 ±1.mantissa × 2^(exponent) 형태의 정규화를 통해 저장됩니다. 32비트 단정도: 부호 1비트 지수 8비트 가수 23비트. 지수는 바이어스(Bias=127)를 더해 저장합니다.

// 표현 범위를 초과할 때 발생하는 오류
오버플로우는 연산 결과가 해당 자료형의 표현 가능한 범위를 벗어날 때 발생합니다. 부호 있는 8비트 정수의 범위는 -128 ~ +127이며, 127+1 = -128이 되어버립니다! 부호 비트의 변화로 오버플로우를 감지합니다.

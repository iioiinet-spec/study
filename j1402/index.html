정올 #1402 약수 구하기

n의 약수를 작은 수부터 찾고, k번째 약수를 출력하는 문제를 직접 실험하며 이해해 봅시다.

1문제 핵심

정수 n과 k가 주어질 때,
n의 약수 중에서 k번째로 작은 약수를 출력합니다.
약수의 개수가 k개보다 적으면 0을 출력합니다.

2인터렉티브 약수 계산기

예시처럼 n = 6, k = 3을 넣어도 되고, 다른 값을 넣어도 됩니다.

버튼을 누르면 약수 목록과 k번째 약수가 표시됩니다.

3초등학생도 이해하는 설명

약수는 어떤 수를 나누었을 때 나머지가 0이 되게 만드는 수입니다.

예를 들어 6은 1, 2, 3, 6으로 나누면 딱 나누어떨어집니다.

그래서 6의 약수는 1, 2, 3, 6이고, 3번째 약수는 3입니다.

4나누어보기 과정

1부터 n까지 차례대로 나누어 보면서 약수인지 확인합니다.

5코드 흐름

약수를 찾는 순간 count를 1씩 증가시키고, count가 k와 같아지면 바로 출력합니다.

#include <stdio.h>

int main() {
    int n, k;
    int count = 0;

    scanf("%d %d", &n, &k);

    for (int i = 1; i <= n; i++) {
        if (n % i == 0) {
            count++;

            if (count == k) {
                printf("%d\n", i);
                return 0;
            }
        }
    }

    printf("0\n");
    return 0;
}

6코드 설명

  1. 1부터 n까지 차례대로 확인합니다.
  2. n % i == 0이면 i는 n의 약수입니다.
  3. 약수를 찾으면 count++로 약수 개수를 증가시킵니다.
  4. count == k가 되면 i가 k번째 약수입니다.
  5. 끝까지 못 찾으면 약수가 부족하므로 0을 출력합니다.

7확인 퀴즈

12의 약수는 1, 2, 3, 4, 6, 12입니다. 그렇다면 4번째 약수는 무엇일까요?

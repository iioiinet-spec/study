1. 문제 이해

참외밭은 ㄱ자 모양의 육각형입니다. 1㎡당 자라는 참외 개수 K와 밭 둘레의 6개 변 정보가 주어질 때, 전체 참외 개수를 구해야 합니다.

방향 번호의미분류
1동쪽가로
2서쪽가로
3남쪽세로
4북쪽세로
방향과 길이가 6개만 주어지므로 복잡한 좌표 계산 없이 큰 직사각형과 작은 직사각형을 찾으면 됩니다.

2. 핵심 아이디어

큰 직사각형가장 긴 가로 × 가장 긴 세로
작은 직사각형안쪽으로 파인 부분의 넓이
참외밭 넓이큰 직사각형 넓이 − 작은 직사각형 넓이
정답참외밭 넓이 × K

2.1 작은 직사각형은 어떻게 찾을까?

가장 긴 가로의 양옆 세로 길이 차이가 작은 직사각형의 세로입니다. 가장 긴 세로의 양옆 가로 길이 차이가 작은 직사각형의 가로입니다.

smallHeight = |가장 긴 가로의 양옆 변 길이 차이|
smallWidth  = |가장 긴 세로의 양옆 변 길이 차이|
6개의 변은 원처럼 이어져 있으므로 이전 인덱스와 다음 인덱스를 구할 때 % 6을 사용합니다.

4. 코드 자세한 설명

4.1 입력 저장

int dir[6];
int len[6];

for (int i = 0; i < 6; i++) {
    scanf("%d %d", &dir[i], &len[i]);
}

방향과 길이를 각각 배열에 저장합니다. 변은 항상 6개입니다.

4.2 가장 긴 가로와 세로 찾기

if (dir[i] == 1 || dir[i] == 2)

방향 1과 2는 가로 방향입니다.

else

방향 3과 4는 세로 방향입니다.

가장 긴 가로와 가장 긴 세로는 큰 직사각형의 가로와 세로가 됩니다.

4.3 원형 인덱스

(index + 5) % 6  // 이전 변
(index + 1) % 6  // 다음 변

6개의 변은 원처럼 연결되어 있으므로 0번 앞은 5번, 5번 뒤는 0번입니다. 그래서 % 6을 사용합니다.

4.4 작은 직사각형 구하기

int smallHeight = abs(len[(maxWidthIndex + 5) % 6] - len[(maxWidthIndex + 1) % 6]);
int smallWidth = abs(len[(maxHeightIndex + 5) % 6] - len[(maxHeightIndex + 1) % 6]);

가장 긴 가로의 양옆 세로 차이가 작은 세로입니다. 가장 긴 세로의 양옆 가로 차이가 작은 가로입니다.

4.5 넓이와 정답

int bigArea = maxWidth * maxHeight;
int smallArea = smallWidth * smallHeight;

printf("%d\n", (bigArea - smallArea) * K);

참외밭 넓이를 구한 뒤 1㎡당 참외 개수 K를 곱합니다.

7. 핵심 정리

  1. 참외밭은 큰 직사각형에서 작은 직사각형을 뺀 모양입니다.
  2. 방향 1, 2는 가로이고 방향 3, 4는 세로입니다.
  3. 가장 긴 가로와 가장 긴 세로가 큰 직사각형의 크기입니다.
  4. 가장 긴 가로의 양옆 길이 차이가 작은 세로입니다.
  5. 가장 긴 세로의 양옆 길이 차이가 작은 가로입니다.
  6. 이전과 다음 인덱스는 % 6으로 원형 처리합니다.
  7. 정답은 (큰 넓이 − 작은 넓이) × K입니다.

8. 정답 C 코드

#include <stdio.h>
#include <stdlib.h>

int main() {
    int K;
    int dir[6];
    int len[6];

    scanf("%d", &K);

    for (int i = 0; i < 6; i++) {
        scanf("%d %d", &dir[i], &len[i]);
    }

    int maxWidth = 0;
    int maxHeight = 0;
    int maxWidthIndex = 0;
    int maxHeightIndex = 0;

    for (int i = 0; i < 6; i++) {
        if (dir[i] == 1 || dir[i] == 2) {
            if (len[i] > maxWidth) {
                maxWidth = len[i];
                maxWidthIndex = i;
            }
        } else {
            if (len[i] > maxHeight) {
                maxHeight = len[i];
                maxHeightIndex = i;
            }
        }
    }

    int smallHeight = abs(len[(maxWidthIndex + 5) % 6] - len[(maxWidthIndex + 1) % 6]);
    int smallWidth = abs(len[(maxHeightIndex + 5) % 6] - len[(maxHeightIndex + 1) % 6]);

    int bigArea = maxWidth * maxHeight;
    int smallArea = smallWidth * smallHeight;

    printf("%d\n", (bigArea - smallArea) * K);

    return 0;
}

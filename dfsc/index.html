DFS 코드 — 한 줄씩 완성하기

문제 → 해석 → 배열 설계 → 함수 한 줄씩 → 시뮬레이션

문제
정점이 1번부터 N번까지 있는 무방향 그래프가 있다.
1번 정점에서 시작하여 DFS 탐색 순서를 출력하시오.
단, 정점 번호가 작은 것부터 먼저 방문해야 한다.
입력 형식
첫째 줄에 정점의 개수 N과 간선의 개수 M이 주어진다.
둘째 줄부터 M개의 줄에 두 정점 a, b가 주어진다.
(a번과 b번이 연결됨)
출력 형식
1번 정점에서 시작한 DFS 방문 순서를 공백으로 출력한다.
입출력 예제
입력
5 5 1 2 1 3 2 4 2 5 3 5
출력
1 2 4 5 3
손으로 해석하기
그래프 연결: 1-2, 1-3, 2-4, 2-5, 3-5. 1번에서 번호 작은 것부터 DFS.
1
1 방문 → 연결: 2, 3 → 작은 2 먼저
2
2 방문 → 연결: 1(방문), 4, 5 → 작은 4 먼저
3
4 방문 → 연결: 2(방문) → 더 갈 곳 없음 → 2로 복귀
4
2로 복귀 → 아직 안 간 5 방문
5
5 방문 → 연결: 2(방문), 3 → 3 방문
6
3 방문 → 연결: 1(방문), 5(방문) → 모두 방문 → 완료

결과: 1 → 2 → 4 → 5 → 3

완성된 DFS 코드 전체
#include <stdio.h> #define MAX 101 // ① 그래프와 방문 배열 int graph[MAX][MAX]; // 인접 행렬 int visited[MAX]; // 방문 여부 int n, m; // 정점 수, 간선 수 void dfs(int now) { // ② 들어오자마자 방문 처리 + 출력 visited[now] = 1; printf("%d ", now); // ③ 연결된 정점을 1번부터 순서대로 확인 for (int next = 1; next <= n; next++) { // ④ 연결 + 미방문 조건 if (graph[now][next] == 1 && visited[next] == 0) { // ⑤ 재귀 호출 = 더 깊이 들어가기 dfs(next); // ← 여기로 돌아올 때 = 자동 백트래킹 } } } int main() { scanf("%d %d", &n, &m); for (int i = 0; i < m; i++) { int a, b; scanf("%d %d", &a, &b); // ⑥ 무방향: 양쪽 다 저장 graph[a][b] = 1; graph[b][a] = 1; } dfs(1); // ⑦ 1번부터 DFS 시작 return 0; }

핵심 6단계: ①배열 선언 → ②방문처리+출력 → ③for 반복 → ④조건 확인 → ⑤재귀 호출 → ⑥무방향 저장. 재귀 함수 자체가 "들어갔다 돌아오기"를 자동으로 만들어줍니다.

자주 하는 실수 5가지
실수 1
방문 처리를 안 하는 실수
visited 없이 DFS하면 1↔2↔1↔2 무한 반복. 반드시 함수 시작 시 visited[now]=1.
실수 2
방문 처리를 너무 늦게 하는 실수
함수에 들어오자마자 해야 합니다. 나중에 하면 중복 방문이 생길 수 있습니다.
실수 3
무방향 그래프를 한쪽만 저장하는 실수
graph[a][b]=1만 하면 b→a 탐색이 안 됩니다. graph[b][a]=1도 필수!
실수 4
작은 번호부터 방문 조건을 놓치는 실수
for(next=1; next<=n; next++)로 순서대로 확인해야 작은 번호부터 방문됩니다.
실수 5
DFS와 BFS를 혼동하는 실수
DFS는 큐가 아닌 재귀(또는 스택) 사용. BFS처럼 while+큐로 짜면 다른 결과가 나옵니다.
이 코드로 도전할 수 있는 확장 과제
시작 정점을 1이 아닌 입력으로 받게 바꾸기
방문 순서 대신 방문 개수를 세도록 바꾸기
연결 요소 개수를 세는 코드로 바꾸기 (미방문 정점마다 DFS 시작)
인접 행렬 대신 인접 리스트로 바꿔보기

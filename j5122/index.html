정올 #5122 조약돌

최소 작업 횟수 DP + 인터랙티브 체험

1. 문제를 아주 쉽게 말하면

여러 칸에 조약돌이 놓여 있습니다. 우리는 모든 조약돌을 없애야 합니다. 단, 행동 횟수를 가장 적게 해야 합니다.

규칙 1: 한 칸 제거

한 칸에서 원하는 만큼 조약돌을 없앨 수 있습니다. 보통 그 칸을 한 번에 비우는 데 사용합니다.

규칙 2: 옆 칸 같이 제거

서로 붙어 있는 두 칸에서 같은 개수만큼 조약돌을 없앨 수 있습니다.

2. 직접 해보기

칸을 클릭한 뒤 버튼을 눌러 보세요. 목표는 모든 조약돌을 가장 적은 횟수로 없애는 것입니다.

조작 버튼

1번 칸부터 클릭해서 시작해 보세요.

3. 핵심 생각

1
왼쪽부터 차례대로 본다.

현재 칸에 조약돌이 남아 있으면, 그 칸을 어떻게 없앨지 결정합니다.

2
혼자 없앨지, 오른쪽 칸과 같이 없앨지 고른다.

옆 칸과 같이 없애면 한 번의 행동으로 두 칸을 줄일 수 있어서 이득일 수 있습니다.

3
가장 적은 행동 횟수를 기억한다.

컴퓨터는 여러 방법을 비교하면서 더 적은 횟수만 남깁니다. 이것이 DP입니다.

4. 예시 이해하기

상태 행동 결과
2 5 3 1번과 2번에서 2개씩 같이 제거 0 3 3
0 3 3 2번과 3번에서 3개씩 같이 제거 0 0 0

총 2번으로 모두 없앨 수 있습니다.

5. C 정답 코드

아래 코드는 상태를 저장하면서 최소 작업 횟수를 찾는 DP 코드입니다.

#include <stdio.h>
#include <stdlib.h>

typedef struct {
    long long remain;
    int cost;
} State;

State dp[3000], ndp[6000];

int cmp(const void *a, const void *b) {
    State *x = (State *)a;
    State *y = (State *)b;

    if (x->remain < y->remain) return -1;
    if (x->remain > y->remain) return 1;
    return x->cost - y->cost;
}

void addState(State arr[], int *cnt, long long remain, int cost) {
    arr[*cnt].remain = remain;
    arr[*cnt].cost = cost;
    (*cnt)++;
}

int main() {
    int N;
    long long a[2505];

    scanf("%d", &N);

    for (int i = 0; i < N; i++) {
        scanf("%lld", &a[i]);
    }

    int dpCnt = 1;
    dp[0].remain = a[0];
    dp[0].cost = 0;

    for (int i = 0; i < N - 1; i++) {
        int ndpCnt = 0;

        for (int j = 0; j < dpCnt; j++) {
            long long cur = dp[j].remain;
            int cost = dp[j].cost;

            if (cur == 0) {
                addState(ndp, &ndpCnt, a[i + 1], cost);
            } else {
                addState(ndp, &ndpCnt, a[i + 1], cost + 1);

                if (cur <= a[i + 1]) {
                    addState(ndp, &ndpCnt, a[i + 1] - cur, cost + 1);
                }
            }
        }

        qsort(ndp, ndpCnt, sizeof(State), cmp);

        dpCnt = 0;

        for (int j = 0; j < ndpCnt; j++) {
            if (dpCnt == 0 || dp[dpCnt - 1].remain != ndp[j].remain) {
                dp[dpCnt++] = ndp[j];
            } else {
                if (ndp[j].cost < dp[dpCnt - 1].cost) {
                    dp[dpCnt - 1].cost = ndp[j].cost;
                }
            }
        }
    }

    int answer = 987654321;

    for (int i = 0; i < dpCnt; i++) {
        int result = dp[i].cost;

        if (dp[i].remain > 0) {
            result++;
        }

        if (result < answer) {
            answer = result;
        }
    }

    printf("%d\n", answer);

    return 0;
}

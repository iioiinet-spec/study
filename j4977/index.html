실수의 이진수

10진수 실수를 입력받아 2진수로 변환하는 문제입니다. 핵심은 정수 부분은 2로 나누기, 소수 부분은 2를 곱하기입니다.

1. 문제 요약

0보다 크고 100보다 작은 실수 N을 입력받아 2진수로 변환합니다.

소수 부분은 소수점 아래 4자리까지 출력합니다.

입력 예시

27.625

출력 예시

11011.1010

2. 실수 나누기

실수는 정수 부분과 소수 부분으로 나누어 생각합니다.

27.625 = 27 + 0.625
정수 부분 27은 나누기, 소수 부분 0.625는 곱하기로 처리합니다.

3. 변환 규칙

  1. 정수 부분은 2로 계속 나누고 나머지를 거꾸로 읽습니다.
  2. 소수 부분은 2를 계속 곱하고 정수 부분을 차례대로 읽습니다.
  3. 소수 부분은 반드시 4자리까지 출력합니다.

4. 핵심 공식

정수 부분: 몫이 0이 될 때까지 2로 나누기
소수 부분: 4번 반복해서 2를 곱하기

5. 인터랙티브 변환기

실수를 입력하면 정수 부분, 소수 부분, 변환 과정, 최종 2진수가 자동으로 계산됩니다.

정수 부분 27
소수 부분 0.625
소수 2진수 4자리 1010
11011.1010

6. 정수 부분 변환 과정

정수 부분은 2로 나누며 나온 나머지를 저장하고, 마지막에 거꾸로 읽습니다.

7. 소수 부분 변환 과정

소수 부분은 2를 곱한 뒤, 정수 부분만 출력하고 남은 소수 부분으로 계속 진행합니다.

단계 계산 출력 비트 다음 소수 부분

8. 코드 작성 순서

  1. 실수 N을 입력받습니다.
  2. 정수 부분은 (int)N으로 구합니다.
  3. 소수 부분은 N - 정수부분으로 구합니다.
  4. 정수 부분은 2로 나누며 나머지를 배열에 저장합니다.
  5. 저장된 나머지를 거꾸로 출력합니다.
  6. 소수 부분은 4번 반복하면서 2를 곱하고 정수 부분을 출력합니다.

9. 정답 C 코드

#include <stdio.h>

int main() {
    double N;
    int intPart;
    double fracPart;

    int binary[100];
    int idx = 0;

    scanf("%lf", &N);

    intPart = (int)N;
    fracPart = N - intPart;

    if (intPart == 0) {
        printf("0");
    } else {
        while (intPart > 0) {
            binary[idx] = intPart % 2;
            idx++;
            intPart = intPart / 2;
        }

        for (int i = idx - 1; i >= 0; i--) {
            printf("%d", binary[i]);
        }
    }

    printf(".");

    for (int i = 0; i < 4; i++) {
        fracPart = fracPart * 2;

        int bit = (int)fracPart;
        printf("%d", bit);

        fracPart = fracPart - bit;
    }

    return 0;
}

10. 코드 자세한 설명

1. 정수 부분과 소수 부분 분리

intPart = (int)N;
fracPart = N - intPart;

예를 들어 N이 27.625라면 intPart는 27, fracPart는 0.625입니다.

2. 정수 부분 2진수 변환

while (intPart > 0) {
    binary[idx] = intPart % 2;
    idx++;
    intPart = intPart / 2;
}

2로 나누었을 때의 나머지를 저장합니다. 이 나머지를 거꾸로 읽으면 2진수가 됩니다.

3. 소수 부분 2진수 변환

for (int i = 0; i < 4; i++) {
    fracPart = fracPart * 2;
    int bit = (int)fracPart;
    printf("%d", bit);
    fracPart = fracPart - bit;
}

소수 부분에 2를 곱한 뒤 정수 부분만 출력합니다. 문제 조건에 맞게 4번 반복합니다.

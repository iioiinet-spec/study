정올 #1828 냉장고

화학 물질들의 보관 온도 범위를 보고, 최소 몇 대의 냉장고가 필요한지 그리디로 찾아봅니다.

1. 문제를 쉽게 이해하기

화학 물질

각 물질은 보관 가능한 최저 온도 ~ 최고 온도 범위가 있습니다.

같은 냉장고 조건

여러 물질의 온도 범위가 겹치면 같은 온도로 맞춘 냉장고에 함께 넣을 수 있습니다.

목표

모든 물질을 보관하기 위해 필요한 냉장고의 최소 개수를 구합니다.

초등학생 버전: 냉장고는 한 가지 온도로만 맞출 수 있습니다. 사과도 5도에서 괜찮고, 우유도 5도에서 괜찮다면 같은 냉장고에 넣을 수 있습니다. 하지만 어떤 물질은 5도에서 안 된다면 새 냉장고가 필요합니다.

2. 직접 입력하고 과정 보기

온도 범위 시각화

최고 온도가 작은 순서로 정렬한 뒤, 현재 냉장고 온도에 들어가지 못하면 새 냉장고를 만듭니다.

최소 냉장고 수

3

현재 냉장고 온도

5

현재 단계

준비

3. 최고 온도 기준 정렬 과정

냉장고 문제는 최고 온도가 작은 순서로 정렬하는 것이 핵심입니다.

순서 최저 온도 low 최고 온도 high 현재 냉장고 온도 판정 설명

4. 왜 최고 온도가 작은 순서로 정렬할까?

  1. 한 냉장고는 한 가지 온도로만 맞출 수 있습니다.
  2. 어떤 물질의 최고 온도가 낮으면 선택할 수 있는 온도 범위가 빨리 끝납니다.
  3. 그래서 최고 온도가 낮은 물질을 먼저 보고, 그 최고 온도에 냉장고를 맞춥니다.
  4. 그 온도에 들어갈 수 있는 다음 물질들은 같은 냉장고에 넣습니다.
  5. 만약 어떤 물질의 최저 온도가 현재 냉장고 온도보다 높으면, 현재 냉장고에는 넣을 수 없으므로 새 냉장고가 필요합니다.

쉽게 말하면, “가장 까다롭게 빨리 끝나는 물질”부터 맞추는 방법입니다.

5. 예제 설명

입력:

5
-15 5
-10 36
10 73
27 44
50 60

최고 온도 기준으로 정렬하면:

-15 ~ 5
-10 ~ 36
27 ~ 44
50 ~ 60
10 ~ 73
  1. 첫 번째 물질은 -15~5입니다. 냉장고 온도를 5도로 맞춥니다.
  2. -10~36은 5도에서 보관 가능하므로 같은 냉장고에 넣습니다.
  3. 27~44는 5도에서 보관할 수 없습니다. 새 냉장고를 만들고 44도로 맞춥니다.
  4. 50~60은 44도에서 보관할 수 없습니다. 새 냉장고를 만들고 60도로 맞춥니다.
  5. 10~73은 60도에서 보관 가능하므로 같은 냉장고에 넣습니다.
  6. 따라서 냉장고는 총 3대입니다.

6. 정답 C 코드

#include <stdio.h>
#include <stdlib.h>

typedef struct {
    int low;
    int high;
} Chemical;

Chemical arr[101];

int compare(const void *a, const void *b) {
    Chemical *x = (Chemical *)a;
    Chemical *y = (Chemical *)b;

    return x->high - y->high;
}

int main() {
    int N;
    int i;

    scanf("%d", &N);

    for (i = 0; i < N; i++) {
        scanf("%d %d", &arr[i].low, &arr[i].high);
    }

    qsort(arr, N, sizeof(Chemical), compare);

    int fridge = 1;
    int temp = arr[0].high;

    for (i = 1; i < N; i++) {
        if (arr[i].low > temp) {
            fridge++;
            temp = arr[i].high;
        }
    }

    printf("%d\n", fridge);

    return 0;
}

최대공약수와 최소공배수

두 자연수를 입력받아 최대공약수와 최소공배수를 구하는 문제입니다. 핵심은 유클리드 호제법과 두 수의 곱 공식입니다.

1. 문제 요약

두 개의 자연수 A, B를 입력받아 최대공약수와 최소공배수를 출력합니다.

입력

24 18

출력

6
72

2. 최대공약수

최대공약수는 두 수를 모두 나누어떨어지게 하는 수 중에서 가장 큰 수입니다.

예: 24와 18의 공통 약수는 1, 2, 3, 6이고, 가장 큰 수는 6입니다.

3. 최소공배수

최소공배수는 두 수의 공통 배수 중에서 가장 작은 수입니다.

예: 24의 배수와 18의 배수 중 처음으로 같은 수는 72입니다.

4. 핵심 공식

최소공배수 = A × B ÷ 최대공약수

예를 들어 A = 24, B = 18, 최대공약수 = 6이면 

최소공배수 = 24 × 18 ÷ 6 = 72

5. 인터랙티브 계산기

숫자를 바꾸면 최대공약수, 최소공배수, 유클리드 호제법 과정이 자동으로 계산됩니다.

최대공약수 6
최소공배수 72

6. 유클리드 호제법 과정

유클리드 호제법은 큰 수를 작은 수로 나눈 나머지를 계속 이용합니다.

gcd(A, B) = gcd(B, A % B)

7. 계산 과정 표

단계 A B A % B 다음 상태

8. 코드 작성 순서

  1. 두 자연수 a, b를 입력받습니다.
  2. 유클리드 호제법으로 최대공약수 g를 구합니다.
  3. a × b ÷ g 공식으로 최소공배수를 구합니다.
  4. 최대공약수와 최소공배수를 한 줄씩 출력합니다.

9. 정답 C 코드

#include <stdio.h>

int gcd(int a, int b) {
    int r;

    while (b != 0) {
        r = a % b;
        a = b;
        b = r;
    }

    return a;
}

int main() {
    int a, b;
    int g, l;

    scanf("%d %d", &a, &b);

    g = gcd(a, b);
    l = a * b / g;

    printf("%d\n", g);
    printf("%d\n", l);

    return 0;
}
코드가 복사되었습니다.

10. 코드 자세한 설명

1. gcd 함수

int gcd(int a, int b) {
    int r;

    while (b != 0) {
        r = a % b;
        a = b;
        b = r;
    }

    return a;
}

b가 0이 될 때까지 나머지를 계속 구합니다. b가 0이 되면 a가 최대공약수입니다.

2. 최소공배수 계산

l = a * b / g;

두 수의 곱은 최대공약수와 최소공배수의 곱과 같습니다. 그래서 두 수의 곱을 최대공약수로 나누면 최소공배수가 됩니다.

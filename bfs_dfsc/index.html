인접 행렬로 DFS/BFS 코드 짜기

그래프 연결 정보를 2차원 배열에 저장 → DFS(재귀) + BFS(큐) 완전 정복

인접 행렬이란?
그래프의 연결 정보를 2차원 배열로 저장하는 방법. graph[a][b]=1이면 a와 b가 연결됨.

그래프 시각화

무방향 그래프 → 행렬이 대각선 기준으로 대칭. graph[a][b] = graph[b][a] = 1

입력 저장 코드
#define MAX 101 int graph[MAX][MAX]; int n, m; scanf("%d %d", &n, &m); for (int i = 0; i < m; i++) { int a, b; scanf("%d %d", &a, &b); graph[a][b] = 1; // a→b graph[b][a] = 1; // b→a (무방향!) }
이웃 정점 탐색 패턴
for (int next = 1; next <= n; next++) { if (graph[now][next] == 1) { ... } }

DFS + BFS 전체 코드
#include <stdio.h> #define MAX 101 int graph[MAX][MAX]; // 인접 행렬 int visited[MAX]; // 방문 배열 int queue[MAX]; // BFS 큐 int front = 0, rear = 0; int n, m; // ─── DFS ───────────────────────────── void dfs(int now) { visited[now] = 1; // ① 방문 처리 printf("%d ", now); // ② 출력 for (int next = 1; next <= n; next++) { if (graph[now][next] == 1 && visited[next] == 0) { dfs(next); // ③ 재귀 호출 } } } // ─── BFS ───────────────────────────── void bfs(int start) { queue[rear++] = start; // ① 시작점 삽입 visited[start] = 1; // ② 방문 처리 while (front < rear) { int now = queue[front++]; // ③ 꺼내기 printf("%d ", now); for (int next = 1; next <= n; next++) { if (graph[now][next] == 1 && visited[next] == 0) { visited[next] = 1; // ④ 방문 처리 queue[rear++] = next; // ⑤ 큐 삽입 } } } } int main() { scanf("%d %d", &n, &m); for (int i = 0; i < m; i++) { int a, b; scanf("%d %d", &a, &b); graph[a][b] = graph[b][a] = 1; } dfs(1); // → 1 2 4 5 3 printf("\n"); // BFS 전에 초기화 필수! for (int i = 1; i <= n; i++) visited[i] = 0; front = rear = 0; bfs(1); // → 1 2 3 4 5 printf("\n"); return 0; }

DFS 후 BFS 전에 반드시: ①visited 초기화 ②front=rear=0. 안 하면 BFS가 작동 안 합니다!

DFS vs BFS 핵심 차이
DFS — 깊이 우선
재귀 함수
방문처리+출력 먼저
dfs(next) 호출
결과: 1 2 4 5 3
BFS — 너비 우선
큐(front/rear)
삽입 시 방문처리
queue[rear++]=next
결과: 1 2 3 4 5

둘 다 인접 행렬을 쓰지만, 다음 정점 처리 방식이 다릅니다. DFS=재귀, BFS=큐.

자주 하는 실수 6가지
실수 1
무방향 그래프를 한쪽만 저장
graph[a][b]=1만 하고 graph[b][a]=1 빠뜨리기.
실수 2
DFS 후 visited 초기화 안 함
BFS 전에 for(i=1;i<=n;i++) visited[i]=0; 필수.
실수 3
BFS 큐 초기화 안 함
front = rear = 0;으로 초기화해야 합니다.
실수 4
DFS 방문 처리를 늦게 함
함수 시작하자마자 visited[now]=1을 해야 합니다.
실수 5
BFS 방문 처리를 늦게 함
큐에 넣을 때 즉시 visited[next]=1. 꺼낼 때 하면 중복 삽입 발생.
실수 6
front/rear 혼동
front=꺼내는 쪽, rear=넣는 쪽. front<rear이면 큐에 원소 있음.

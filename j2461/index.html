정올 #2461 공주님의 정원

3월 1일부터 11월 30일까지 꽃이 끊기지 않도록 최소 개수의 꽃을 고르는 그리디 문제입니다.

1. 문제를 자세히 이해하기

목표 기간

정원에는 3월 1일부터 11월 30일까지 매일 꽃이 피어 있어야 합니다.

꽃 정보

각 꽃은 피는 날짜와 지는 날짜가 있습니다.

피는 월 피는 일 지는 월 지는 일

정답

조건을 만족하도록 고를 수 있는 꽃의 최소 개수를 출력합니다. 불가능하면 0입니다.

중요: 꽃이 지는 날은 꽃이 피어 있지 않습니다. 따라서 11월 30일까지 피어 있으려면 마지막 꽃의 지는 날짜가 적어도 12월 1일이어야 합니다.

2. 날짜를 숫자로 바꾸는 방법

월과 일을 따로 비교하면 복잡하므로 날짜를 하나의 숫자로 바꿉니다.

3월 1일  → 301
5월 10일 → 510
11월 30일 → 1130
12월 1일 → 1201

코드에서는 다음 함수를 사용합니다.

int toDate(int month, int day) {
    return month * 100 + day;
}

이렇게 바꾸면 301보다 401이 더 크고, 1130보다 1201이 더 크므로 날짜 비교가 쉬워집니다.

3. 그리디 핵심 생각

  1. 현재 꽃이 이어져야 하는 날짜를 target이라고 합니다.
  2. 처음 target은 3월 1일, 즉 301입니다.
  3. target 날짜 이전 또는 target 날짜에 피기 시작한 꽃만 선택할 수 있습니다.
  4. 그중에서 가장 늦게 지는 꽃을 고릅니다.
  5. 고른 꽃의 지는 날짜로 target을 늘립니다.
  6. target이 1201 이상이 되면 11월 30일까지 덮은 것이므로 성공입니다.
  7. 만약 target을 더 늘릴 수 없다면 중간에 꽃이 끊긴 것이므로 실패입니다.

초등학생 버전: 꽃 릴레이입니다. 지금 날짜까지 이어졌다면, 그 날짜 전에 이미 피어 있는 꽃 중에서 가장 오래 버티는 꽃을 골라 다음 꽃에게 바통을 넘깁니다.

4. 인터랙티브로 과정 보기

날짜 타임라인

꽃 기간

선택된 꽃 릴레이

그리디 실행을 누르면 target 날짜가 어떻게 늘어나는지 볼 수 있습니다.

첫 줄 N, 이후 N줄에 꽃의 시작 월/일, 종료 월/일을 입력합니다.

최소 꽃 개수

2

현재 target

6. 소스 작성 방법

  1. Flower 구조체를 만듭니다. 시작 날짜 start와 종료 날짜 end를 저장합니다.
  2. toDate() 함수로 월/일을 숫자 날짜로 바꿉니다.
  3. 꽃들을 시작 날짜 오름차순으로 정렬합니다. 시작 날짜가 같으면 종료 날짜가 늦은 꽃을 앞에 둡니다.
  4. target = 301, finish = 1201로 시작합니다.
  5. 현재 target까지 피기 시작한 꽃들을 모두 확인합니다.
  6. 그 꽃들 중 가장 늦게 지는 날짜 maxEnd를 찾습니다.
  7. maxEnd가 target과 같으면 더 이상 이어갈 수 없으므로 0을 출력합니다.
  8. 이어갈 수 있다면 꽃 하나를 선택한 것이므로 answer를 1 증가시키고 target = maxEnd로 바꿉니다.
  9. target이 1201 이상이 될 때까지 반복합니다.

7. 왜 이 방법이 맞을까?

현재 target 날짜가 비어 있으면 안 됩니다. 따라서 다음 꽃은 반드시 target 날짜 이전 또는 target 날짜에 이미 피어 있어야 합니다.

그중 가장 늦게 지는 꽃을 고르면, 같은 꽃 1개를 쓰면서 가장 멀리까지 기간을 늘릴 수 있습니다.

즉, 지금 고를 수 있는 꽃 중에서 가장 멀리 보내 주는 꽃을 고르는 것이 항상 이득입니다. 그래서 그리디로 풀 수 있습니다.

8. 정답 C 코드

#include <stdio.h>
#include <stdlib.h>

#define MAXN 100001

typedef struct {
    int start;
    int end;
} Flower;

Flower flower[MAXN];

int compare(const void *a, const void *b) {
    Flower *x = (Flower *)a;
    Flower *y = (Flower *)b;

    if (x->start != y->start) {
        return x->start - y->start;
    }

    return y->end - x->end;
}

int toDate(int month, int day) {
    return month * 100 + day;
}

int main() {
    int N;
    int sm, sd, em, ed;
    int i;

    scanf("%d", &N);

    for (i = 0; i < N; i++) {
        scanf("%d %d %d %d", &sm, &sd, &em, &ed);

        flower[i].start = toDate(sm, sd);
        flower[i].end = toDate(em, ed);
    }

    qsort(flower, N, sizeof(Flower), compare);

    int target = 301;
    int finish = 1201;
    int answer = 0;
    int idx = 0;

    while (target < finish) {
        int maxEnd = target;

        while (idx < N && flower[idx].start <= target) {
            if (flower[idx].end > maxEnd) {
                maxEnd = flower[idx].end;
            }
            idx++;
        }

        if (maxEnd == target) {
            printf("0\n");
            return 0;
        }

        answer++;
        target = maxEnd;
    }

    printf("%d\n", answer);

    return 0;
}

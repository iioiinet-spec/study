정올 #1457 영역 구하기

직사각형이 칠해진 모눈종이에서 남은 빈 영역의 개수와 각 영역의 넓이를 DFS/BFS로 찾아봅니다.

1. 문제를 쉽게 이해하기

모눈종이

M×N 크기의 격자가 있습니다. 각 칸은 빈 칸이거나 직사각형으로 칠해진 칸입니다.

영역

빈 칸끼리 위, 아래, 왼쪽, 오른쪽으로 연결되어 있으면 하나의 영역입니다.

정답

빈 영역의 개수와 각 영역의 넓이를 오름차순으로 출력합니다.

초등학생 버전: 색칠된 칸은 벽이고, 하얀 칸은 빈 땅입니다. 서로 붙어 있는 빈 땅 덩어리가 몇 개인지, 각 덩어리의 칸 수가 몇 개인지 세는 문제입니다.

2. 예제 입력으로 직접 확인하기

빈 칸 직사각형 탐색 중 현재 칸

빈 칸을 클릭하면 직접 벽으로 바꾸거나 다시 빈 칸으로 바꿀 수 있습니다.

영역 개수

3

넓이 오름차순

1 7 13

현재 단계

준비

3. 현재 탐색 결과

순서 현재 칸 영역 번호 현재 영역 넓이 설명

4. 좌표 처리 핵심

입력으로 직사각형의 왼쪽 아래 점과 오른쪽 위 점이 주어집니다.

입력: x1 y1 x2 y2
칠하는 칸: x1 <= x < x2, y1 <= y < y2
  1. x는 열 번호로 생각합니다. 그래서 배열에서는 c가 됩니다.
  2. y는 행 번호로 생각합니다. 그래서 배열에서는 r이 됩니다.
  3. x2와 y2는 포함하지 않습니다. 따라서 반복문은 r < y2, c < x2까지입니다.
  4. 예를 들어 0 2 4 4라면 x는 0,1,2,3이고 y는 2,3입니다. 총 4×2=8칸을 칠합니다.

5. DFS/BFS 설명

  1. 격자를 왼쪽 위부터 차례대로 봅니다.
  2. 아직 방문하지 않은 빈 칸을 찾으면 새로운 영역이 시작됩니다.
  3. 그 칸에서 위, 아래, 왼쪽, 오른쪽으로 갈 수 있는 빈 칸을 계속 찾습니다.
  4. 갈 수 있는 칸이 없으면 하나의 영역 탐색이 끝납니다.
  5. 그 영역에서 방문한 칸의 개수가 넓이입니다.
  6. 모든 영역을 찾은 뒤 넓이를 오름차순으로 정렬합니다.

6. 정답 C 코드

#include <stdio.h>

int M, N, K;
int map[101][101];
int area[10001];
int areaCount = 0;

int dr[4] = {-1, 1, 0, 0};
int dc[4] = {0, 0, -1, 1};

int dfs(int r, int c) {
    int i;
    int size = 1;

    map[r][c] = 1;

    for (i = 0; i < 4; i++) {
        int nr = r + dr[i];
        int nc = c + dc[i];

        if (nr >= 0 && nr < M && nc >= 0 && nc < N) {
            if (map[nr][nc] == 0) {
                size += dfs(nr, nc);
            }
        }
    }

    return size;
}

void sortArea() {
    int i, j, temp;

    for (i = 0; i < areaCount - 1; i++) {
        for (j = i + 1; j < areaCount; j++) {
            if (area[i] > area[j]) {
                temp = area[i];
                area[i] = area[j];
                area[j] = temp;
            }
        }
    }
}

int main() {
    int i, j;
    int x1, y1, x2, y2;
    int r, c;

    scanf("%d %d %d", &M, &N, &K);

    for (i = 0; i < K; i++) {
        scanf("%d %d %d %d", &x1, &y1, &x2, &y2);

        for (r = y1; r < y2; r++) {
            for (c = x1; c < x2; c++) {
                map[r][c] = 1;
            }
        }
    }

    for (i = 0; i < M; i++) {
        for (j = 0; j < N; j++) {
            if (map[i][j] == 0) {
                area[areaCount] = dfs(i, j);
                areaCount++;
            }
        }
    }

    sortArea();

    printf("%d\n", areaCount);

    for (i = 0; i < areaCount; i++) {
        printf("%d ", area[i]);
    }

    printf("\n");

    return 0;
}

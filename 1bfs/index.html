문제 1
그래프 BFS 방문 순서
정점 N개, 간선 M개. 1번 정점에서 BFS를 수행했을 때 방문 순서를 출력하시오.
입력 예제
5 4 1 2 1 3 2 4 3 5
출력 예제
1 2 3 4 5
단계별 시뮬레이터
그래프: 1-2, 1-3, 2-4, 3-5
정점 상태
큐 상태 (앞→뒤)
단계 0
시작 버튼을 눌러 BFS를 시작하세요.
범례: ■ 현재  ■ 방문  ■ 큐 대기  ■ 미방문
정답 코드
#include <stdio.h> #define MAX 101 int graph[MAX][MAX], visited[MAX]; int queue[MAX], front=0, rear=0; int n, m; void bfs(int start) { queue[rear++] = start; visited[start] = 1; while (front < rear) { int now = queue[front++]; printf("%d ", now); for (int i=1; i<=n; i++) { if (graph[now][i] && !visited[i]) { visited[i] = 1; queue[rear++] = i; } } } } int main() { scanf("%d %d", &n, &m); for (int i=0; i<m; i++) { int a, b; scanf("%d %d", &a, &b); graph[a][b] = graph[b][a] = 1; } bfs(1); return 0; }

핵심: 큐에 넣을 때 visited=1로 방문 처리. front<rear이면 큐가 비어 있지 않음.

문제 2
특정 정점까지 거리
1번 정점에서 각 정점까지의 최단 거리를 출력하시오.
입력 예제
5 4 1 2 1 3 2 4 3 5
출력 예제
0 1 1 2 2
거리 배열 시뮬레이터
그래프: 1-2, 1-3, 2-4, 3-5. dist[next] = dist[now] + 1
dist 배열
큐 상태
단계 0
시작 버튼을 눌러 최단거리 BFS를 시작하세요.
정답 코드
int dist[MAX]; // dist[1]=0 (시작점) void bfs(int start) { queue[rear++] = start; visited[start] = 1; // dist[start] = 0 (전역 초기화) while (front < rear) { int now = queue[front++]; for (int i=1; i<=n; i++) { if (graph[now][i] && !visited[i]) { visited[i] = 1; dist[i] = dist[now] + 1; // 핵심! queue[rear++] = i; } } } }

dist[i] = dist[now] + 1 — 처음 방문했을 때가 최단거리. BFS가 거리 순서로 탐색하기 때문입니다.

문제 3
도달 가능 여부
정점 A에서 B로 갈 수 있으면 1, 없으면 0 출력.
입력 예제
5 4 1 2 2 3 3 4 4 5 1 5
출력 예제
1
핵심 아이디어
1
시작점 A에서 BFS를 실행한다
2
큐에서 꺼낸 정점이 B이면 즉시 1을 반환한다
3
BFS가 끝날 때까지 B를 만나지 못하면 0을 반환한다

또는 단순히 BFS 후 visited[B]가 1이면 도달 가능, 0이면 불가능으로도 판단할 수 있습니다.

정답 코드
int bfs(int start, int target) { queue[rear++] = start; visited[start] = 1; while (front < rear) { int now = queue[front++]; if (now == target) return 1; // 발견! for (int i=1; i<=n; i++) { if (graph[now][i] && !visited[i]) { visited[i] = 1; queue[rear++] = i; } } } return 0; // 도달 불가 } int main() { int a, b; /* 입력 생략 */ scanf("%d %d", &a, &b); printf("%d", bfs(a, b)); }
문제 4
미로 최단거리
0은 벽, 1은 길. (0,0)에서 (n-1,m-1)까지 최소 이동 횟수를 구하시오.
입력 예제
3 3 111 010 111
출력 예제
4
미로 BFS 시뮬레이터
클릭으로 벽/길 토글. 시작(0,0) 파랑, 끝(우하) 주황.
클릭 = 벽/길 토글
dist 값 (격자)
BFS 실행을 눌러보세요.
정답 코드
int dx[4]={-1,1,0,0}; int dy[4]={0,0,-1,1}; void bfs() { qx[rear]=0; qy[rear]=0; rear++; dist[0][0]=1; // 시작칸 포함 while(front<rear) { int x=qx[front], y=qy[front++]; for(int i=0; i<4; i++) { int nx=x+dx[i], ny=y+dy[i]; if(nx<0||ny<0||nx>=n||ny>=m) continue; if(map[nx][ny]=='0') continue; if(dist[nx][ny]) continue; // 방문 체크 dist[nx][ny]=dist[x][y]+1; qx[rear]=nx; qy[rear]=ny; rear++; } } }

dist[0][0]=1로 시작칸 포함. dist!=0으로 방문 여부 체크. 4방향 탐색은 dx,dy 배열로.

문제 5
섬의 개수 (BFS)
1은 땅, 0은 물. 상하좌우로 연결된 땅 덩어리(섬)의 개수를 구하시오.
섬 개수 시뮬레이터
클릭으로 땅/물 토글. BFS 실행 후 섬마다 다른 색으로 표시됩니다.
땅을 클릭해 배치하고 BFS 실행을 눌러보세요.
정답 코드 핵심
void bfs(int x, int y) { // 한 섬 전체를 BFS로 방문 front=rear=0; qx[rear]=x; qy[rear]=y; rear++; visited[x][y]=1; while(front<rear) { int cx=qx[front], cy=qy[front++]; for(int i=0;i<4;i++){ int nx=cx+dx[i], ny=cy+dy[i]; if(범위 초과) continue; if(map[nx][ny]==0||visited[nx][ny]) continue; visited[nx][ny]=1; qx[rear]=nx; qy[rear]=ny; rear++; } } } int main() { int count=0; /* 입력 */ for(int i=0;i<n;i++) for(int j=0;j<m;j++) if(map[i][j]==1 && !visited[i][j]){ bfs(i,j); count++; // 새 섬 발견! } printf("%d",count); }

"방문 안 한 땅을 만날 때마다 BFS 시작 = 새 섬 하나 발견" — 연결 요소 개수 세기 패턴입니다.

C언어 fgets 인터렉티브 강의자료

공백 포함 문자열 입력 · 줄바꿈 제거 · scanf와 함께 쓰기 · strtok 단어 분리

1. fgets란?

fgets는 C언어에서 한 줄 전체를 입력받을 때 사용하는 함수입니다.

1. 공백이 포함된 문장을 입력받을 수 있습니다.

2. 배열 크기를 함께 전달하므로 scanf("%s")보다 안전합니다.

3. 줄바꿈 문자 \\n까지 저장될 수 있습니다.

char line[100];

fgets(line, sizeof(line), stdin);

2. scanf("%s")와 fgets 차이 체험

아래 문장을 입력하면 scanf("%s")와 fgets가 어떻게 다르게 저장하는지 볼 수 있습니다.

3. fgets 줄바꿈 문자 체험

fgets는 엔터를 누르면 줄바꿈 문자도 함께 저장할 수 있습니다. 아래에서 줄바꿈 제거 전후를 확인하세요.

line[strcspn(line, "\n")] = '\0';

4. 단어 분리 strtok 체험

fgets로 문장 한 줄을 입력받은 뒤, strtok로 공백 기준 단어를 나눌 수 있습니다.

5. scanf 뒤 fgets 문제 체험

scanf로 숫자를 입력받은 뒤 바로 fgets를 쓰면, 숫자 뒤에 남은 줄바꿈 문자를 fgets가 읽어버릴 수 있습니다.

잘못된 코드

scanf("%d", &n);
fgets(line, sizeof(line), stdin);

숫자 뒤의 엔터가 남아 있어서 line이 빈 문자열처럼 처리될 수 있습니다.

해결 코드

scanf("%d", &n);
getchar();
fgets(line, sizeof(line), stdin);

getchar()로 남은 엔터를 제거한 뒤 fgets를 실행합니다.

6. 직접 문제 풀이 예제

예제 1. 문장 그대로 출력

#include <stdio.h>

int main() {
    char sentence[100];

    fgets(sentence, sizeof(sentence), stdin);

    printf("%s", sentence);

    return 0;
}

예제 2. 줄바꿈 제거 후 출력

#include <stdio.h>
#include <string.h>

int main() {
    char sentence[100];

    fgets(sentence, sizeof(sentence), stdin);

    sentence[strcspn(sentence, "\n")] = '\0';

    printf("[%s]\n", sentence);

    return 0;
}

예제 3. 단어 개수 세기

#include <stdio.h>
#include <string.h>

int main() {
    char sentence[200];
    int count = 0;

    fgets(sentence, sizeof(sentence), stdin);

    sentence[strcspn(sentence, "\n")] = '\0';

    char *token = strtok(sentence, " ");

    while (token != NULL) {
        count++;
        token = strtok(NULL, " ");
    }

    printf("%d\n", count);

    return 0;
}

7. 인터렉티브 단어 개수 세기

문장을 입력하면 strtok처럼 공백 기준으로 단어를 나누고 개수를 세어 줍니다.

8. 자주 하는 실수

실수 1. scanf("%s")로 문장 입력받기

scanf("%s")는 공백 전까지만 입력받습니다.

입력이 hello world라면 scanf("%s")는 hello만 저장합니다. 문장 전체가 필요하면 fgets를 써야 합니다.

실수 2. 줄바꿈 문자를 제거하지 않기

fgets는 \\n을 포함할 수 있어서 strcmp 결과가 달라질 수 있습니다.

입력 END를 fgets로 받으면 실제 저장값이 END\\n일 수 있습니다. strcmp(line, "END")가 실패할 수 있으므로 줄바꿈을 제거해야 합니다.

실수 3. scanf 다음에 바로 fgets 사용

남은 엔터를 fgets가 읽을 수 있습니다.

scanf("%d", &n); 뒤에 getchar();를 넣어 남은 줄바꿈을 제거하면 해결됩니다.

9. fgets 기본 공식

#include <stdio.h>
#include <string.h>

int main() {
    char line[1000];

    fgets(line, sizeof(line), stdin);

    line[strcspn(line, "\n")] = '\0';

    printf("%s\n", line);

    return 0;
}

10. 핵심 정리

1. fgets는 한 줄 전체를 입력받습니다.

2. 공백이 포함된 문장 입력에 좋습니다.

3. 줄바꿈 문자 \\n이 저장될 수 있습니다.

4. 줄바꿈 제거는 strcspn을 사용합니다.

5. scanf 뒤 fgets를 쓸 때는 getchar로 엔터를 제거합니다.

6. 단어 분리는 strtok와 함께 사용합니다.
